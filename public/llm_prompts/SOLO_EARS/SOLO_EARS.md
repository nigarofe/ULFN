# Structured Outcomes of Learning Objectives (SOLO) described using EARS patterns

## Pre-structural
When the response
- doesn't attack the task properly
- has irrelevant comments or misses the point entirely
- shows a lack of understanding of the point
- takes too simple an approach
- misses the point and shows little evidence of relevant learning
the system shall classify the response as 'Pre-structural'

When the prompt
- contains a question that requires basic engagement but elicits no relevant response
- contains a simple task whose core demand is not addressed
the system shall classify the question as 'Pre-structural'

## Uni-structural
When the response
- focuses on one relevant aspect without further elaboration
- gives slightly relevant but vague answers that lack depth
- identifies a single key element but ignores connections to others
- shows the ability to identify and name a few things and follow simple procedures
the system shall classify the response as 'Uni-structural'

When the prompt
- targets a single aspect using verbs like identify or name
- asks for recall of one key fact or element
- contains a task that calls for basic definition or recognition without requiring links
- contains a question that involves simple procedures or identification following taught steps
the system shall classify the question as 'Uni-structural'

## Multi-structural
When the response
- focuses on several relevant aspects but treats them independently and additively
- lists multiple ideas quantitatively without synthesis or qualitative depth
- shows knowledge in disconnected tidbits but fails to present or explain relationships
- shows knowledge at the level of remembering, memorizing, and parroting
the system shall classify the response as 'Multi-structural'

When the prompt
- contains a question that requires listing multiple elements using verbs like list or describe
- contains a task that involves additive knowledge accumulation, like classifying items separately
- contains a question that is a surface-level recall question focusing on quantity over integration
- asks to gather several facts without expecting connections
the system shall classify the question as 'Multi-structural'

## Relational
When the response
- integrates different aspects into a coherent whole
- demonstrates an adequate understanding by linking ideas meaningfully
- identifies various patterns and views the topic from distinct perspectives
- explains links, compares and contrasts elements, and uses systemic modeling
- compares, contrasts, and explains relationships between ideas
the system shall classify the response as 'Relational'

When the prompt
- contains a question that encourages connections using verbs like compare, contrast, or explain
- contains a prompt that asks the learner to integrate aspects into a coherent explanation
- contains a task that requires analysis of relationships and multiple perspectives
- contains a question that involves linking ideas or systemic modeling of the topic
the system shall classify the question as 'Relational'

## Extended abstract
When the response
- conceptualizes the integrated whole at a higher level of abstraction
- generalizes the understanding to a new topic or area
- applies the concepts creatively in real-life or novel contexts
- goes beyond given material to create new knowledge and apply it in multiple contexts
the system shall classify the response as 'Extended abstract'

When the prompt
- incentivizes generalization using verbs like hypothesize or create
- requires applying knowledge creatively to novel or real-life situations
- asks the learner to extend beyond the material to theorize or invent
- contains a question that is abstract and requires prediction or synthesis in new contexts
the system shall classify the question as 'Extended abstract'
