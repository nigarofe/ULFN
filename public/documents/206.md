# Proposition
Solve the following differential equations:

a) $6x^2y'' + 9xy' - 3y = 0$

b) $x^2y'' + 5xy' + 4y = 0$

c) $x^2y'' + xy' + y = 0$

Observation: The alternatives are the examples, not the questions 7-10, because the questions themselves were repetitive.

# Step-by-step
## a
$$6x^2y'' + 9xy' - 3y = 0$$
$$6 r(r-1) + 9r - 3 = 0$$
$$6r^2 + 3r - 3 = 0$$
$$2r^2 + r - 1 = 0$$
$$\Delta = b^2 - 4ac = 1^2 - 4 (2) (-1) = 9$$
$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-1 \pm 3}{4} = \frac{1}{2}, -1$$
$$y(x) = c_1 x^{r_1} + c_2 x^{r_2} = c_1x^{\frac{1}{2}} + c_2x^{-1}$$

## b
$$x^2y'' + 5xy' + 4y = 0$$
$$r(r-1) + 5r + 4 = 0$$
$$r^2 + 4r + 4 = 0$$
$$\Delta = b^2 - 4ac = 4^2 - 4(1)(4) = 0$$
$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-4 \pm 0}{2} = -2$$
$$y(x) = c_1 x^{r} + c_2 \ln(x) x^{r}  = c_1x^{-2} + c_2 \ln(x) x^{-2}$$

## c
$$x^2y'' + xy' + y = 0$$
$$r(r-1) + r + 1 = 0$$
$$r^2 + 1 = 0$$
$$r = \pm \sqrt{-1}$$
$$r = \pm i$$
$$y(x) = c_1 \cos(\ln|x|) + c_2 \sin(\ln|x|)$$

# Answers
a) $c_1x^{\frac{1}{2}} + c_2x^{-1}$

b) $c_1x^{-2} + c_2 \ln(x) x^{-2}$

c) $c_1 \cos(\ln|x|) + c_2 \sin(\ln|x|)$
