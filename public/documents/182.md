# Proposition
Solve the following equations:

a) $\frac{dy}{dx} = \frac{x^5}{y^3}$, where $y(0) = 3$
b) $\frac{dy}{dx} = \frac{4x^3 + 6x^2}{y^2}$, where $y(1) = 3$
c) $\frac{dy}{dx} = (1-2x)y^2$, where $y(0) = 1$
d) $1- e^{-x} \frac{dy}{dx} = 0$, where $y(0) = 2$

Solved items:
- a, b
- c, d
- a, b

# Step-by-step

# Answer

b) $y = \left( 3x^4 + 6x^3 + 18 \right)^{1/3}$
c) $y = \left( x^2 - x + 1 \right)^{-1}$
d) $y = \left(e^x + 1\right)$*
