# Proposition
Solve the following differential equation:

$$(ye^{xy} \cos(2x) - 2e^{xy} \sin(2x) + 2x) + (xe^{xy} \cos(2x) - 3) y' = 0$$

# Step-by-step
$$(y e^{xy} \cos(2x) - 2 e^{xy} \sin(2x) + 2x) + (x e^{xy} \cos(2x) - 3) y' = 0$$

$$M = y e^{xy} \cos(2x) - 2 e^{xy} \sin(2x) + 2x$$

$$N = x e^{xy} \cos(2x) - 3$$

$$M_y = \cos(2x) \frac{\partial}{\partial y} y e^{xy} - 2 \sin(2x) \frac{\partial}{\partial y} e^{xy}$$

$$M_y = \cos(2x) [y e^{xy} x + 1 e^{xy}] - 2 \sin(2x) [e^{xy} x]$$

$$M_y = e^{xy} (y x \cos(2x) + \cos(2x) - 2 x \sin(2x))$$

$$N_x = \frac{\partial}{\partial x} (x e^{xy}) (\cos(2x)) = [ (x e^{xy}) (-2\sin(2x)) + (x e^{xy} y + 1 e^{xy}) (\cos(2x)) ]$$

$$N_x = e^{xy} (y x \cos(2x) + \cos(2x) - 2 x \sin(2x))$$

Como $M_y = N_x$, a equação é exata. Portanto existe uma função $f$ tal que $f_x = M$ e $f_y = N$

$$\int f_y dy = \int N dy$$

$$f = \int x e^{xy} \cos(2x) - 3 dy = x \cos(2x) \int e^{xy} dy - 3 \int dy$$

$$u = xy \quad \frac{du}{dy} = x \Rightarrow dy = \frac{1}{x} du$$

$$\int e^{xy} dy = \frac{1}{x} \int e^u du = \frac{1}{x} e^{xy}$$

$$f = x \cos(2x) \left( \frac{1}{x} e^{xy} \right) - 3y + h(x) = e^{xy} \cos(2x) - 3y + h(x)$$

$$f_x = M$$

$$\frac{\partial}{\partial x} (e^{xy} \cos(2x) - 3y + h(x)) = y e^{xy} \cos(2x) - 2 e^{xy} \sin(2x) + 2x$$

$$\frac{\partial}{\partial x} (e^{xy} \cos(2x)) + h'(x) = y e^{xy} \cos(2x) - 2 e^{xy} \sin(2x) + 2x$$

$$(e^{xy}) (-2 \sin(2x)) + (y e^{xy}) (\cos(2x)) + h'(x) = \dots$$

$$h'(x) = 2x \Rightarrow h(x) = x^2$$

$$\boxed{e^{xy} \cos(2x) - 3y + x^2 = c}$$

# Answer
$$\boxed{e^{xy} \cos(2x) - 3y + x^2 = c}$$
