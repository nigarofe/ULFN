# Proposition
Find the Laplace transform of the following functions:

a) $f(t) = 1$
b) $f(t) = e^{at}$

**Bonus (optional as resolution takes time)**

c) $f(t) = \sin (at)$

Solved items:

- a
- b
- a
- b
- a

# Step-by-step
## General
$$F(s) = \int_\alpha^\beta K(s,t) f(t) dt$$
$$F(s) = \int_0^\infty e^{-st} f(t) dt$$
$$\mathcal{L}\{f(t)\} = F(s) = \int_0^\infty e^{-st} f(t) dt$$

## b
$$\mathcal{L}\{e^{at}\} = F(s) = \int_0^\infty e^{-st} e^{at} dt$$
$$\int_0^\infty e^{t(a-s)} dt$$
$$u = t(a-s) \implies \frac{du}{dt} = (a-s) \implies dt = \frac{1}{a-s} du$$
$$\frac{1}{a-s} \int_0^\infty e^u du = \frac{1}{a-s} \left[ e^{t(a-s)} \right]_0^\infty$$

Se definirmos que $s > a$, entĂŁo $\lim_{t \to \infty} e^{t(a-s)} = 0$

$$\frac{1}{a-s} \left[ e^{t(a-s)} \right]_0^\infty = \frac{1}{a-s} [0 - 1] = \frac{1}{s-a}$$
$$\mathcal{L}\{e^{at}\} = \frac{1}{s-a} \qquad s>a$$

## c
$$\mathcal{L}\{\sin(at)\} = F(s) = \color{red} \int_0^\infty e^{-st} \sin(at) dt$$
$$F(s) = \color{blue} \frac{1}{a} - \frac{s}{a} \int_0^\infty e^{-st} \cos(at) dt$$
$$F(s) = \frac{1}{a} - \frac{s}{a} \color{green} \left[ \frac{s}{a} F(s) \right]$$
$$F(s) = \frac{1}{a} - \frac{s^2}{a^2} F(s)$$
$$F(s) + \frac{s^2}{a^2} F(s) = \frac{1}{a}$$
$$F(s) \left( 1 + \frac{s^2}{a^2} \right) = \frac{1}{a}$$
$$F(s) \left( \frac{a^2 + s^2}{a^2} \right) = \frac{1}{a}$$
$$F(s) = \frac{1}{a} \cdot \frac{a^2}{s^2 + a^2}$$
$$F(s) = \frac{a}{s^2 + a^2}$$
$$\mathcal{L}\{\sin(at)\} = \frac{a}{s^2 + a^2} \quad s>0$$



### Integration by parts of $\color{red} \int_0^\infty e^{-st} \sin(at) dt$

$$\int fg' dt = fg - \int f'g dt$$
$$f = e^{-st} \quad g' = \sin(at)$$
Chain rule:
$$f' = \frac{d}{dt}(e^{-st}) = \frac{d}{du} e^u \cdot \frac{d}{dt}(-st) = -s e^{-st}$$
Integration by substitution:
$$g = \int \sin(at) dt$$
$$u = at \implies \frac{du}{dt} = a \implies dt = \frac{1}{a} du$$
$$g = \int \sin(u) \cdot \frac{1}{a} du = \frac{1}{a} \int \sin(u) du = -\frac{1}{a} \cos(at)$$
$$\left[\left(e^{-st}\right) \left(-\frac{1}{a} \cos(at)\right)\right]_0^\infty - \int_0^\infty (-s e^{-st}) \left(-\frac{1}{a} \cos(at)\right) dt$$
$$-\frac{1}{a} \left[e^{-st} \cos(at)\right]_0^\infty - \frac{s}{a} \int_0^\infty e^{-st} \cos(at) dt$$
$$-\frac{1}{a} \left[0 - 1\right] - \frac{s}{a} \int_0^\infty e^{-st} \cos(at) dt$$
$$
\color{blue}{\boxed{
\frac{1}{a} - \frac{s}{a} \int_0^\infty e^{-st} \cos(at) dt
}}
$$

### Integration by parts of $\int_0^\infty e^{-st} \cos(at) dt$
$$\int fg' dt = fg - \int f'g dt$$
$$f = e^{-st} \quad g' = \cos(at)$$
Chain rule:
$$f' = \frac{d}{dt}(e^{-st}) = \frac{d}{du} e^u \cdot \frac{d}{dt}(-st) = -s e^{-st}$$
Integration by substitution:
$$g = \int \cos(at) dt$$
$$u = at \implies \frac{du}{dt} = a \implies dt = \frac{1}{a} du$$
$$g = \int \cos(u) \cdot \frac{1}{a} du = \frac{1}{a} \int \cos(u) du = \frac{1}{a} \sin(at)$$
$$\left[\left(e^{-st}\right) \left(\frac{1}{a} \sin(at)\right)\right]_0^\infty - \int_0^\infty (-s e^{-st}) \left(\frac{1}{a} \sin(at)\right) dt$$
$$\frac{1}{a} \left[\left(e^{-st}\right) \left(\sin(at)\right)\right]_0^\infty + \frac{s}{a} \int_0^\infty e^{-st} \sin(at) dt$$
$$\frac{1}{a} \left[0 - 0\right] + \frac{s}{a} \int_0^\infty e^{-st} \sin(at) dt$$
$$\frac{s}{a}  \color{red} \int_0^\infty e^{-st} \sin(at) dt$$
$$
\color{green}{\boxed{
\left[ \frac{s}{a} F(s) \right]
}} 
$$

---

# Answers
a) $\mathcal{L}\{1\} = \frac{1}{s} \quad s>0$

b) $\mathcal{L}\{e^{at}\} = \frac{1}{s-a} \quad s>a$

c) $\mathcal{L}\{\sin(at)\} = \frac{a}{s^2 + a^2} \quad s>0$
