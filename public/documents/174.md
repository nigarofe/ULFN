# Proposition
Solve for y in these differential equations:

a) $\frac{dy}{dx} = 8x$, where $y(0) = 4$

b) $\frac{dy}{dx} = 2x + 2$, where $y(0) = 2$

c) $\frac{dy}{dx} = 6x + 5$, where $y(0) = 10$

d) $\frac{dy}{dx} = 8x + 3$, where $y(0) = 12$

Solved items: 
- a, b, c, d
- a, b
- c, d
- a, b

# Step-by-step

# Answers

a) $y = 4x^2 + 4$

b) $y = x^2 + 2x + 2$

c) $y = 3x^2 + 5x + 10$

d) $y = 4x^2 + 3x + 12$
