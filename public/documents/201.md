# Proposition
Solve the following differential equations by using a series solution:

a) $$\frac{d^2y}{dx^2} + 4y = 0$$

b) $$\frac{d^2y}{dx^2} - y = 0$$

Solved items:

- a
- a
- a
- b
- a
- b

# Step-by-step
> See document 246 for a similar problem.

## General
$$y = \sum_{n=0}^{\infty} a_n x^n$$
$$y' = \sum_{n=0}^{\infty} (n+1) a_{n+1} x^n$$
$$y'' = \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} x^n$$

## a
$$\frac{d^2y}{dx^2} + 4y = 0$$

---

$$y = \sum_{n=0}^{\infty} a_n x^n = a_0 + a_1 x + a_2 x^2 + a_3 x^3 + a_4 x^4 + \cdots$$
$$y' = a_1 + 2a_2 x + 3a_3 x^2 + 4a_4 x^3 + \cdots$$
$$y' = \sum_{n=1}^{\infty} n a_n x^{n-1} = \sum_{n=0}^{\infty} (n+1) a_{n+1} x^n$$
$$y'' = 2a_2 + 6a_3 x + 12a_4 x^2 + \cdots$$
$$y'' = \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2} = \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} x^n$$

---

$$\sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} x^n + 4 \sum_{n=0}^{\infty} a_n x^n = 0$$
$$\sum_{n=0}^{\infty} \left[(n+2)(n+1) a_{n+2} + 4 a_n\right] x^n = 0$$

Como a série é nula para todo $x$:
$$(n+2)(n+1) a_{n+2} + 4 a_n = 0$$ 

---

$$n = 0 \implies (2)(1) a_2 + 4 a_0 = 0$$
$$\implies a_2 = - \frac{4a_0}{(2)(1)}$$
$$n = 2 \implies (4)(3) a_4 + 4 a_2 = 0$$
$$\implies a_4 = - \frac{4a_2}{(4)(3)} = \frac{4^2 a_0}{(4)(3)(2)(1)}$$
$$a_n = a_{2m} = (-1)^m \frac{4^m a_0}{(2m)!} \quad m = 0, 1, 2, \ldots$$

---

$$n=1 \implies (3)(2) a_3 + 4 a_1 = 0$$
$$\implies a_3 = - \frac{4a_1}{(3)(2)}$$
$$n=3 \implies (5)(4) a_5 + 4 a_3 = 0$$
$$\implies a_5 = - \frac{4a_3}{(5)(4)} = \frac{4^2 a_1}{(5)(4)(3)(2)}$$
$$a_n = a_{2m+1} = (-1)^m \frac{4^m a_1}{(2m+1)!} \quad m = 0, 1, 2, \ldots$$

---

$$y = a_0 \sum_{m=0}^{\infty} (-1)^m \frac{4^m x^{2m}}{(2m)!}  + a_1 \sum_{m=0}^{\infty} (-1)^m \frac{4^m x^{2m+1}}{(2m+1)!} $$
$$y = a_0 \sum_{m=0}^{\infty} (-1)^m \frac{(2x)^{2m}}{(2m)!}  + \frac{a_1}{2} \sum_{m=0}^{\infty} (-1)^m \frac{(2x)^{2m+1}}{(2m+1)!} $$
$$y = a_0 \cos(2x) + \frac{a_1}{2} \sin(2x)$$

## b
$$y'' - y = 0$$

$$y = \sum_{n=0}^\infty a_n x^n = a_0 x^0 + a_1 x^1 + a_2 x^2 + a_3 x^3 + a_4 x^4 + \dots$$

$$y' = a_1 + 2a_2 x + 3a_3 x^2 + 4a_4 x^3 + \dots = \sum_{n=1}^\infty n a_n x^{n-1} = \sum_{n=0}^\infty (n+1) a_{n+1} x^n$$

$$y'' = 2a_2 + 6a_3 x + 12a_4 x^2 + \dots = \sum_{n=2}^\infty (n-1) n a_n x^{n-2} = \sum_{n=0}^\infty (n+1)(n+2) a_{n+2} x^n$$

$$\left( \sum_{n=0}^\infty (n+1)(n+2) a_{n+2} x^n \right) - \left( \sum_{n=0}^\infty a_n x^n \right) = 0$$

$$\sum_{n=0}^\infty [(n+1)(n+2) a_{n+2} - a_n] x^n = 0$$

$$(n+1)(n+2) a_{n+2} - a_n = 0$$

$$n=0 \rightarrow (1)(2) a_2 - a_0 = 0 \rightarrow a_2 = \frac{a_0}{(2)(1)}$$

$$n=2 \rightarrow (3)(4) a_4 - a_2 = 0 \rightarrow a_4 = \frac{a_2}{(4)(3)} = \frac{a_0}{(4)(3)(2)(1)}$$

$$a_n = a_{2m} = \frac{a_0}{(2m)!}$$

$$n=1 \rightarrow (2)(3) a_3 - a_1 = 0 \rightarrow a_3 = \frac{a_1}{(3)(2)}$$

$$n=3 \rightarrow (4)(5) a_5 - a_3 = 0 \rightarrow a_5 = \frac{a_3}{(5)(4)} = \frac{a_1}{(5)(4)(3)(2)}$$

$$a_n = a_{2m+1} = \frac{a_1}{(2m+1)!}$$

$$y(x) = a_0 \sum_{m=0}^\infty \frac{x^{2m}}{(2m)!} + a_1 \sum_{m=0}^\infty \frac{x^{(2m+1)}}{(2m+1)!}$$

$$y(x) = c_0 e^x + c_1 e^{-x}$$

# Answers
a) $y = a_0 \cos(2x) + \frac{a_1}{2} \sin(2x)$

b) $y = c_0 e^x + c_1 e^{-x}$
