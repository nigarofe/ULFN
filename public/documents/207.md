# Proposition
Determine the Euler equation that this differential equation is similar to; then determine the Euler equation's two roots, $r_1$ and $r_2$:

a) $4x^2 y'' - 2x y' + 2(1+x^2) y = 0$

b) $2x^2 y'' - 3x y' + (2 + 2x + x^2) y = 0$

Solved items:
- a
- a
- a
- b
- a
- b

# Step-by-step

## General
$$\color{blue} x^2 y'' + p_0 \, xy' + q_0 \, y = 0$$
$$\color{blue} x^2 y'' + \left[x \, p(x) \right] xy' + \left[x^2 \, q(x) \right] y = 0$$
$$\color{blue} p_0 = \lim_{x \to 0} \left[x \, p(x) \right]$$
$$\color{blue} q_0 = \lim_{x \to 0} \left[x^2 \, q(x) \right]$$

## a
$$4x^2 y'' - 2x y' + 2(1+x^2) y = 0$$

$$\color{blue} x^2 y'' + \left[x \, p(x) \right] xy' + \left[x^2 \, q(x) \right] y = 0$$

$$\frac{4}{4}x^2 y'' - \frac{2}{4}x y' + \frac{2}{4}(1+x^2) y = 0$$

$$x^2 y'' - \frac{1}{2} x y' + \left(\frac{1}{2} + \frac{x^2}{2}\right) y = 0$$

$$p(x) = -\frac{1}{2x}$$

$$q(x) = \frac{1}{2x^2} + \frac{1}{2}$$

$$\color{blue} x^2 y'' + p_0 \, xy' + q_0 \, y = 0$$

$${\color{blue} p_0 = \lim_{x \to 0} \left[x \, p(x) \right]} = \lim_{x \to 0} \left(-\frac{1}{2}\right) = -\frac{1}{2}$$

$${\color{blue} q_0 = \lim_{x \to 0} \left[x^2 \, q(x) \right]} = \lim_{x \to 0} \left(\frac{1}{2} + \frac{x^2}{2}\right) = \frac{1}{2}$$

$$x^2 y'' - \frac{1}{2} x y' + \frac{1}{2} y = 0$$

$$r(r-1) - \frac{1}{2} r + \frac{1}{2} = 0$$

$$r^2 - r - \frac{1}{2} r + \frac{1}{2} = 0$$

$$r^2 - \frac{3r}{2} + \frac{1}{2} = 0$$

$$2r^2 - 3r + 1 = 0$$

$$\Delta = b^2 - 4ac = (-3)^2 - 4(2)(1) = 1$$

$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-(-3) \pm \sqrt{1}}{2(2)} = \frac{3 \pm 1}{4}$$

$$r_1 = 1, \quad r_2 = \frac{1}{2}$$

## b
$$2 x^2 y'' - 3 x y' + (2 + 2x + x^2) y = 0$$

$$x^2 y'' - \frac{3}{2} x y' + \frac{1}{2} (2 + 2x + x^2) y = 0$$

$$\color{blue} x^2 y'' + \left[x \, p(x) \right] xy' + \left[x^2 \, q(x) \right] y = 0$$

$$p(x) = -\frac{3}{2x}$$

$$q(x) = \frac{1}{2} + \frac{1}{x} + \frac{1}{x^2}$$

$$\color{blue} x^2 y'' + p_0 \, xy' + q_0 \, y = 0$$

$${\color{blue} p_0 = \lim_{x \to 0} \left[x \, p(x) \right]} = \lim_{x \to 0} -\frac{3}{2} = -\frac{3}{2}$$

$${\color{blue} q_0 = \lim_{x \to 0} \left[x^2 \, q(x) \right]} = \lim_{x \to 0} \frac{x^2}{2} + x + 1 = 1$$

$$x^2 y'' - \frac{3}{2} x y' + y = 0$$

$$2 x^2 y'' - 3 x y' + 2 y = 0$$

$$2 r(r-1) - 3 r + 2 = 0$$

$$2 r^2 - 5 r + 2 = 0$$

$$\Delta = b^2 - 4 a c = (-5)^2 - 4 \cdot 2 \cdot 2 = 9$$

$$\frac{-b \pm \sqrt{\Delta}}{2 a} = \frac{-(-5) \pm \sqrt{9}}{2 \cdot 2} = \frac{5 \pm 3}{4} = 2, \frac{1}{2}$$

# Answers
a) $x^2 y'' - \frac{x}{2} y' + \frac{y}{2} = 0 \quad \text{roots } = 1, \frac{1}{2}$

b) $2x^2 y'' - 3x y' + 2y = 0 \quad \text{roots } = 2, \frac{1}{2}$
