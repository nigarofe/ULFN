# Proposition
Solve the following differential equations:

a) $4y'' + 12y' + 9y = 0$
b) $y'' - 6y' + 13y = 0$
c) $y'' + y' - 2y = t^2$

# Step-by-step
## Geral
Resumo de soluções de $ay'' + by' + cy = 0$

- Raízes reais distintas: $y = c_1 e^{r_1 t} + c_2 e^{r_2 t}$
- Raízes reais iguais: $y = c_1 e^{rt} + c_2 t e^{rt}$
- Raízes complexas: $y = e^{\alpha t}(c_1 \cos(\beta t) + c_2 i \sin(\beta t))$

## a
$$4y'' + 12y' + 9y = 0$$

$$y'' + 3y' + \frac{9}{4}y = 0$$

$$r^2 + 3r + \frac{9}{4} = 0$$

$$\Delta = b^2 - 4ac = 3^2 - 4 \cdot 1 \cdot \frac{9}{4} = 9 - 9 = 0$$

$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-3 \pm \sqrt{0}}{2 \cdot 1} = -\frac{3}{2}$$

$$\boxed{y = c_1 e^{-\frac{3}{2}t} + c_2 t e^{-\frac{3}{2}t}}$$


## b
$$y'' - 6y' + 13y = 0$$

$$r^2 - 6r + 13 = 0$$

$$\Delta = b^2 - 4ac = (-6)^2 - 4 \cdot 1 \cdot 13 = 36 - 52 = -16$$

$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-(-6) \pm \sqrt{-16}}{2 \cdot 1} = \frac{6 \pm 4i}{2} = 3 \pm 2i \implies \alpha = 3, \beta = 2$$

$$y = e^{\alpha t} (c_1 \cos(\beta t) + c_2 \sin(\beta t))$$

$$\boxed{y = e^{3t} (c_1 \cos(2t) + c_2 \sin(2t))}$$


## c
$$y'' + y' - 2y = t^2$$

$$r^2 + r - 2 = 0$$

$$\Delta = b^2 - 4ac = 1^2 - 4 \cdot 1 \cdot (-2) = 1 + 8 = 9$$

$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-1 \pm \sqrt{9}}{2 \cdot 1} = \frac{-1 \pm 3}{2} = 1, -2$$

$$y_H = c_1 e^t + c_2 e^{-2t}$$

$$y_P = At^2 + Bt + C \quad \quad y'_P = 2At + B \quad \quad y''_P = 2A$$

$$(2A) + (2At + B) - 2(At^2 + Bt + C) = t^2$$

$$(-2A)t^2 + (2A - 2B)t + (2A + B - 2C) = t^2$$

$$
\begin{cases}
-2A = 1 \\
2A - 2B = 0 \\
2A + B - 2C = 0
\end{cases}
\implies
A = -\frac{1}{2} \quad B = -\frac{1}{2} \quad C = -\frac{3}{4}
$$

$$y_P = -\frac{1}{2}t^2 - \frac{1}{2}t - \frac{3}{4}$$

$$\boxed{y = c_1 e^t + c_2 e^{-2t} - \frac{1}{2}t^2 - \frac{1}{2}t - \frac{3}{4}}$$

# Answer
## a
$$\boxed{y = c_1 e^{-\frac{3}{2}t} + c_2 t e^{-\frac{3}{2}t}}$$

## b
$$\boxed{y = e^{3t}(c_1 \cos(2t) + c_2 i \sin(2t))}$$

## c
$$\boxed{y = c_1 e^{-2t} + c_2 e^{t} - \frac{1}{2}t^2 - \frac{1}{2}t - \frac{3}{4}}$$
