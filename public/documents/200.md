# Proposition
Shift the following series index to start at n = 0:

a) $$\sum_{n=2}^{\infty} (-1)^n (x-1)^n (x-2)^n$$

b) $$\sum_{n=2}^{\infty} (-1)^n (x-2)^{2n}$$

c) $$\sum_{n=2}^{\infty} (4x+9)(x-2)^{4n}$$

d) $$\sum_{n=3}^{\infty} (x-1)^n(x-2)^{2n} (x-3)^{-n}$$

# Step-by-step
$$y = \sum_{n=2}^{\infty} a_n (x+1)^n$$
$$n=n+2$$
$$y = \sum_{n+2=2}^{\infty} a_{n+2} (x+1)^{n+2}$$
$$y = \sum_{n=0}^{\infty} a_{n+2} (x+1)^{n+2}$$

# Answers
a) $$\sum_{n=0}^{\infty} (-1)^{n+2} (x-1)^{n+2} (x-2)^{n+2}$$

b) $$\sum_{n=0}^{\infty} (-1)^{n+2} (x-2)^{2(n+2)}$$

c) $$\sum_{n=0}^{\infty} (4x+9)(x-2)^{4(n+2)}$$

d) $$\sum_{n=0}^{\infty} (x-1)^{n+3} (x-2)^{2(n+3)} (x-3)^{-(n+3)}$$
