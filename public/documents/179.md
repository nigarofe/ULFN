# Proposition
Find the implicit solution to these differential equations:

a) $(y^2+y) \frac{dy}{dx} - x = 0$
b) $(y^3 + y^2 + y) \frac{dy}{dx} - x^2 - x = 0$
c) $\sin(y) \frac{dy}{dx} - \cos(x) = 0$
d) $(e^y + 3y^2) \frac{dy}{dx} -2x = 0$

Solved items:
- a, b, c, d
- a, b
- c, d

# Step-by-step


# Answer
a) $\frac{y^3}{3} + \frac{y^2}{2} = \frac{x^2}{2} + C$
b) $\frac{y^4}{4} + \frac{y^3}{3} + \frac{y^2}{2} = \frac{x^3}{3} + \frac{x^2}{2} + C$
c) $\cos(y) = \sin(x) + C$
d) $e^y + y^3 = x^2 + C$
