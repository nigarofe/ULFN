# Proposition
Solve the following differential equations:

a) $2x + y^2 + 2xy y' = 0$

b) $(y \cos x + 2 x e^y) + (\sin x + x^2 e^y - 1) y' = 0$

c) $(3xy + y^2) + (x^2 + xy) y' = 0$


# Step-by-step
## a
$$2x + y^2 + 2xyy' = 0$$

$$\frac{\partial M}{\partial y} = 2y \quad \frac{\partial N}{\partial x} = 2y$$

Como $M_y = N_x$, a equação é exata. Portanto existe uma função $f$ tal que $\frac{\partial f}{\partial x} = M$ e $\frac{\partial f}{\partial y} = N$

$$\frac{\partial f}{\partial x} = M$$

$$\int \frac{\partial}{\partial x} f dx = \int M dx$$

$$f = x^2 + xy^2 + g(y)$$

$$\frac{\partial f}{\partial y} = N$$

$$\frac{\partial}{\partial y} (x^2 + xy^2 + g(y)) = 2xy$$

$$2xy + g'(y) = 2xy$$

$$g'(y) = 0$$

$$g(y) = 0$$

$$f = x^2 + xy^2$$

$$\boxed{x^2 + xy^2 = c}$$

## b
$$(y \cos x + 2x e^y) + (\sin x + x^2 e^y - 1)y' = 0$$

$$\frac{\partial M}{\partial y} = \cos x + 2x e^y$$

$$\frac{\partial N}{\partial x} = \cos x + 2x e^y$$

Como $M_y = N_x$, a equação é exata, portanto existe uma função $f$ tal que $\frac{\partial f}{\partial x} = M$ e $\frac{\partial f}{\partial y} = N$

$$\int \frac{\partial f}{\partial x} dx = \int M dx$$

$$f = y \sin x + x^2 e^y + g(y)$$

$$\frac{\partial f}{\partial y} = N$$

$$\frac{\partial}{\partial y} (y \sin x + x^2 e^y + g(y)) = \sin x + x^2 e^y - 1$$

$$\sin x + x^2 e^y + g'(y) = \sin x + x^2 e^y - 1$$

$$g'(y) = -1$$

$$g(y) = -y$$

$$f = y \sin x + x^2 e^y - y$$

$$\boxed{y \sin x + x^2 e^y - y = c}$$

## c
$$(3xy + y^2) + (x^2 + xy)y' = 0$$

$$\frac{\partial M}{\partial y} = 3x + 2y \qquad \frac{\partial N}{\partial x} = 2x + y$$

$$\frac{M_y - N_x}{N} = \frac{(3x + 2y) - (2x + y)}{(x^2 + xy)} = \frac{3x + 2y - 2x - y}{x^2 + xy} = \frac{x + y}{x(x + y)} = \frac{1}{x}$$

Como $M_y \neq N_x$, a equação não é exata. Entretanto, como $\frac{M_y - N_x}{N}$ depende apenas de $x$, existe $\mu$ tal que ela se torna exata.

$$\mu = e^{\int \frac{M_y - N_x}{N} dx} = e^{\int \frac{1}{x} dx} = e^{\ln x} = x$$

$$(3xy + y^2)x + (x^2 + xy)xy' = 0$$

$$(3x^2y + xy^2) + (x^3 + x^2y)y' = 0$$

$$\frac{\partial M}{\partial y} = 3x^2 + 2xy \qquad \frac{\partial N}{\partial x} = 3x^2 + 2xy$$

Como $M_y = N_x$, a equação é exata. Portanto existe uma função $f$ tal que $\frac{\partial f}{\partial x} = M$ e $\frac{\partial f}{\partial y} = N$.

$$\int \frac{\partial f}{\partial x} dx = \int M dx$$

$$f = x^3y + \frac{1}{2}x^2y^2 + g(y)$$

$$\frac{\partial f}{\partial y} = N$$

$$\frac{\partial}{\partial y} \left(x^3y + \frac{1}{2}x^2y^2 + g(y)\right) = x^3 + x^2y$$

$$x^3 + x^2y + g'(y) = x^3 + x^2y$$

$$g'(y) = 0 \implies g = 0$$

$$\boxed{x^3y + \frac{1}{2}x^2y^2 = c}$$

# Answer
## a
$$\boxed{x^2 + xy^2 = c}$$

## b
$$\boxed{y \sin x + e^y x^2 - y = c}$$

## c
$$\boxed{x^3 y + \frac{1}{2} x^2 y^2 = c}$$
