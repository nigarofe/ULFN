# Introdução
**Tipos de esforços internos**
- M = Momento fletor
- T = Momento torçor 
- N = Esforço normal (ou axial)
- V = Esforço cortante (ou transversal)

**Tipos de equilíbrio**
- Equilíbrio externo: cálculo das reações de apoio
- Equilíbrio interno: cálculo dos esforços internos solicitantes (M, T, N, V)

**Esforços por tipo de análise**
- 2D: 1M, 1N, 1V
- 3D: 2M, 1T, 1N, 2V

# Tipos de diagramas
- DCA = Diagrama de Cargas Aplicadas
- DCL = Diagrama de Corpo Livre
- DEC = Diagrama de Esforços Cortantes
- DMF = Diagrama de Momentos Fletores
- DEA = Diagrama de Esforços Axiais
