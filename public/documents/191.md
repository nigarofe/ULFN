# Proposition
Solve the following differential equations:

a) $y'' + 5y' + 6y = 10 \sin x$, where $y(0) = 1$ and $y'(0) = -4$
b) $y'' + 4y' + 3y = 5 \cos(x)$, where $y(0) = \frac{7}{2}$ and $y'(0) = -6$

Solved items:

- a
- b
- a
- b
- a

# Step-by-step
## a
$$y'' + 5y' + 6y = 10 \sin x \quad y(0) = 1 \quad y'(0) = -4$$

---

$$r^2 + 5r + 6 = 0$$
$$\Delta = b^2 - 4ac = 25 - 4(1)(6) = 25 - 24 = 1$$
$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-5 \pm \sqrt{1}}{2(1)} = \frac{-5 \pm 1}{2} = -2, -3$$
$$y_H = c_1e^{r_1x} + c_2e^{r_2x}$$
$$y_H = c_1e^{-2x} + c_2e^{-3x}$$

---

$$y_P = A \sin x + B \cos x$$
$$y_P' = A \cos x - B \sin x$$
$$y_P'' = -A \sin x - B \cos x$$

$$(-A \sin x - B \cos x) + 5(A \cos x - B \sin x) + 6(A \sin x + B \cos x) = 10 \sin x$$
$$-A \sin x - B \cos x + 5A \cos x - 5B \sin x + 6A \sin x + 6B \cos x = 10 \sin x$$
$$(-A - 5B + 6A) \sin x + (-B + 5A + 6B) \cos x = 10 \sin x$$
$$(5A - 5B) \sin x + (5A + 5B) \cos x = (10) \sin x + (0) \cos x$$

$$
\left[
\begin{array}{cc|r}
  5 & -5 & 10 \\
  5 & 5 & 0
\end{array}
\right]
\implies
A = 1
\implies
B = -1
$$

$$y_P = \sin x - \cos x$$

---

$$y = y_H + y_P = c_1e^{-2x} + c_2e^{-3x} + \sin x - \cos x$$
$$y(0) = c_1 + c_2 - 1 = 1 \implies c_1 + c_2 = 2$$
$$y' = -2c_1e^{-2x} - 3c_2e^{-3x} + \cos x + \sin x$$
$$y'(0) = -2c_1 - 3c_2 + 1 = -4 \implies -2c_1 - 3c_2 = -5$$
$$
\left[
\begin{array}{cc|r}
  1 & 1 & 2 \\
  -2 & -3 & -5
\end{array}
\right]
\implies
c_2 = 1, c_1 = 1
$$

---

$$y = e^{-2x} + e^{-3x} + \sin x - \cos x$$

# Answer
a) $y = e^{-2x} + e^{-3x} + \sin(x) - \cos(x)$
b) $y = e^{-x} + 2e^{-3x} + \sin(x) + \frac{\cos(x)}{2}$
