# Proposition
Solve the following differential equations:

a) $y'' + 4y' + 4y = 0$, where $y(0) = 1$ and $y'(0) = 0$.
b) $y'' + 10y' + 25y = 0$, where $y(0) = 1$ and $y'(0) = 2$
c) $y'' + 8y' + 16y = 0$, where $y(0) = 2$ and $y'(0) = 4$
d) $y'' + 6y' + 9y = 0$, where $y(0) = 4$ and $y'(0) = 4$

Solved items: 

- a, b
- c
- d
- a
- b

# Step-by-step
## a
$$y'' + 4y' + 4y = 0 \quad y(0) = 1 \quad y'(0) = 0$$
$$r^2 + 4r + 4 = 0$$
$$\Delta = b^2 - 4ac = 4^2 - 4(1)(4) = 16 - 16 = 0$$
$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-4 \pm \sqrt{0}}{2(1)} = \frac{-4}{2} = -2$$
$$y(x) = c_1xe^{rx} + c_2e^{rx}$$
$$y(x) = c_1xe^{-2x} + c_2e^{-2x}$$
$$y(0) = c_2 = 1 \implies c_2 = 1$$
$$y(x) = c_1xe^{-2x} + e^{-2x}$$
$$y'(x) = ...$$

---

$$\frac{d}{dx} xe^{-2x} = ...$$
$$\frac{d}{dx} fg = fg' + f'g$$
$$f = x \quad f' = 1$$
$$g = e^{-2x} \quad g' = -2e^{-2x}$$
$$\frac{d}{dx} xe^{-2x} = - 2xe^{-2x} + e^{-2x} $$

---

$$y'(x) = c_1(-2xe^{-2x} + e^{-2x}) -2e^{-2x}$$
$$y'(0) = c_1 - 2 = 0 \implies c_1 = 2$$
$$y(x) = 2xe^{-2x} + e^{-2x}$$


# Answer
a) $y(x) = 2xe^{-2x} + e^{-2x}$
b) $y(x) = 7xe^{-5x} + e^{-5x}$
c) $y(x) = 12xe^{-4x} + 2e^{-4x}$
d) $y(x) = 16xe^{-3x} + 4e^{-3x}$
