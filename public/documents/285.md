# Proposition
Calcule a reação vertical no apoio B.

![](../media/285_1.jpg){width=50%}

Descrição: viga de comprimento L com 3 apoios, sendo eles uma apoio articulado em A, um apoio móvel em B e C. O apoio móvel em B está no comprimento L/2. Há uma carga distribuída de w em toda a viga.


# Notes
## Introdução
Queremos calcular as reações de apoio para uma estrutura hiperestática. Para isso, primeiro determinamos o GIE, que é a quantidade de apoios que precisam de ser removidos para que a estrutura torne-se isostática. Essa nova estrutura é chamada de EIF.

## Siglas
- GIE: Grau de Indeterminação Estática
- EIF: Estrutura Isostática Fundamental

- $Q_i$: redundantes hiperestáticos
- $F_{ij}$: flexibilidade
- $D_{QLi}$: deslocamento correspondente à $Q_i$ e causado pelo carregamento na EIF

## Procedimento
1. Obter o GIE e o EIF
2. Fase L: Calcular os deslocamentos $D_{QL1}$ e $D_{QL2}$ 
3. Fase 1: Calcular os deslocamentos $F_{11}$ e $F_{21}$
4. Fase 2: Calcular os deslocamentos $F_{12}$ e $F_{22}$
5. Formular as equações de compatibilidade $[D_{QL}] + [F][Q] = [D_Q]$
6. Resolver o sistema de equações
7. Calcular as reações de apoio via superposição de efeitos


### O Significado das Matrizes na Equação

A equação $[D_{QL}] + [F][Q] = [D_Q]$ é a **Equação de Compatibilidade de Deslocamentos**. Como nós removemos apoios para transformar a estrutura em isostática (o EIF), precisamos garantir que a estrutura final se comporte como a original. Essa equação garante que a soma dos deslocamentos criados pelas cargas e pelas reações redundantes seja igual ao deslocamento real do apoio.

* **$[D_{QL}]$ (Vetor de Deslocamentos das Cargas Externas):** Representa os deslocamentos que ocorrem na Estrutura Isostática Fundamental (EIF) causados *apenas* pelas cargas reais (peso, vento, etc.) aplicadas na estrutura. A letra "L" geralmente vem de *Load* (Carga).
* **$[F]$ (Matriz de Flexibilidade):** É uma matriz quadrada que contém os **coeficientes de flexibilidade**. Ela diz o quanto a estrutura se deforma quando aplicamos forças unitárias (forças de valor 1) nos lugares dos apoios que removemos.
* **$[Q]$ (Vetor de Incógnitas Hiperestáticas):** São as forças ou momentos reativos que você "liberou" para tornar a estrutura isostática. O objetivo do sistema de equações é justamente descobrir o valor dessas forças. *(Nota: Em muitos livros, essa matriz é chamada de $[X]$).*
* **$[D_Q]$ (Vetor de Deslocamentos Prescritos):** São os deslocamentos reais que existem nos apoios da estrutura original. Na grande maioria dos casos, os apoios são rígidos e não se movem, então **essa matriz é geralmente composta por zeros ($[0]$)**. Só terá valores diferentes de zero se houver um recalque de apoio conhecido.

### O que significam $F_{11}, F_{12}, F_{21}$ e $F_{22}$?

Esses são os **Coeficientes de Flexibilidade**. Eles seguem uma nomenclatura padrão baseada em índices $i$ e $j$, escritos como $F_{ij}$:

* O primeiro número ($i$) indica **ONDE** (em qual direção/coordenada) o deslocamento está sendo medido.
* O segundo número ($j$) indica **QUEM** causou aquele deslocamento (onde a carga unitária foi aplicada).

Traduzindo para o seu procedimento:

* **$F_{11}$:** É o deslocamento na direção 1 causado por uma força unitária aplicada na própria direção 1.
* **$F_{12}$:** É o deslocamento na direção 1 causado por uma força unitária aplicada lá na direção 2.
* **$F_{21}$:** É o deslocamento na direção 2 causado por uma força unitária aplicada lá na direção 1.
* **$F_{22}$:** É o deslocamento na direção 2 causado por uma força unitária aplicada na própria direção 2.

> **Dica de Ouro:** Pelo Teorema de Maxwell-Betti (Teorema da Reciprocidade), a matriz de flexibilidade é sempre simétrica. Isso significa que **$F_{12}$ será sempre igual a $F_{21}$**. Você só precisa calcular um deles na prática, o que poupa muito tempo na prova!

**Resumo da sua lógica de cálculo:**
Você calcula o quanto a estrutura deforma com as cargas reais ($D_{QL}$), depois calcula o quanto ela deforma com forças unitárias ($F$). Juntando tudo no sistema linear, você descobre quais forças exatas ($Q$) precisam existir naqueles apoios para "empurrar" a estrutura de volta para a posição original ($D_Q = 0$).




# Step-by-step

Note que $|D_{QL1}| = |D_{Q1}|$. O "L" adicional no $D_{QL1}$ significa "load", indicando que o deslocamento foi causado pelas cargas reais na estrutura. Enquanto $D_{Q1}$ representa o deslocamento causado pelo redundante hiperestático, o qual é igual à reação de apoio em B.



![](../media/285_3.jpg){width="80%"}

$$\left(\rightarrow +\right) \sum F_x = H_A = 0$$

$$\left(\uparrow +\right) \sum F_y = V_A + V_C - wL = 0$$

$$V_A + V_C = wL$$

$$\left(\circlearrowleft +\right) \sum M_A = + V_C \cdot L - (wL)\left(\frac{L}{2}\right) = 0$$

$$V_C L = \frac{wL^2}{2}$$

$$V_C = \frac{wL}{2}$$

$$V_A = wL - V_C = wL - \frac{wL}{2}$$

$$V_A = \frac{wL}{2}$$

---

$$M_L(x) = \frac{wL}{2}x - (wx)\left(\frac{x}{2}\right)$$

$$M_L = \frac{wL}{2}x - \frac{wx^2}{2}$$

---

$$M_U = \begin{cases} -\frac{1}{2}x \text{ para } 0 < x < \frac{L}{2} \\ -\frac{1}{2}x + 1\left(x - \frac{L}{2}\right) \text{ para } \frac{L}{2} < x < L \end{cases}$$

$$M_U = \begin{cases} -\frac{1}{2}x \text{ para } 0 < x < \frac{L}{2} \\ \frac{x}{2} - \frac{L}{2} \text{ para } \frac{L}{2} < x < L \end{cases}$$


---




**Cálculo de $D_{QL1}$ da fase 0**

Aplicando a equação do método da carga unitária para vigas

$$P_U \cdot D_{QL1} = \sum_{i=1}^{m} \left( \int_{L} \frac{M_U M_L}{EI} dx \right)_i = \frac{1}{EI} \int_{L} M_U M_L dx$$

$$= \frac{1}{EI} \left[ \int_{0}^{\frac{L}{2}} \left( -\frac{1}{2} x \right) \left( \frac{wL}{2} x - \frac{wx^2}{2} \right) dx + \int_{\frac{L}{2}}^{L} \left( \frac{x}{2} - \frac{L}{2} \right) \left( \frac{wL}{2} x - \frac{wx^2}{2} \right) dx \right]$$

$$= \frac{2}{EI} \left[ \int_{0}^{\frac{L}{2}} \left( -\frac{1}{2} x \right) \left( \frac{wL}{2} x - \frac{wx^2}{2} \right) dx \right]$$

$$D_{QL1} = - \frac{5 w L^4}{384 EI}$$

---

**Cálculo de $D_{Q1}$, a partir de $F_{11}$ da fase 1**

Como tanto a fase L quanto a fase U da fase 1 são idênticas à fase U da fase 0, podemos reaproveitar a fórmula.

$$
M_U = M_L = \begin{cases}
-\frac{1}{2} x & \text{para } 0 < x < \frac{L}{2} \\
\frac{x}{2} - \frac{L}{2} & \text{para } \frac{L}{2} < x < L
\end{cases}
$$

Aplicando a equação do método da carga unitária para vigas

$$P_U \cdot F_{11} = \sum_{i=1}^{m} \left( \int_{L} \frac{M_U M_L}{EI} dx \right)_i = \frac{1}{EI} \int_{L} M_U^2 dx$$

$$= \frac{1}{EI} \left[ \int_{0}^{\frac{L}{2}} \left( -\frac{1}{2} x \right)^2 dx + \int_{\frac{L}{2}}^{L} \left( \frac{x}{2} - \frac{L}{2} \right)^2 dx \right]$$

$$= \frac{2}{EI} \left[ \int_{0}^{\frac{L}{2}} \left( -\frac{1}{2} x \right)^2 dx \right] = \frac{L^3}{48 EI}$$

---

$$D_{Q1} = Q_1 \cdot F_{11} = \frac{Q_1 \cdot L^3}{48 EI}$$

---

$$|D_{QL1}| = |D_{Q1}|$$

$$\frac{5 w L^4}{384 EI} = \frac{Q_1 L^3}{48 EI}$$

$$Q_1 = \frac{5 w L}{8} = V_B$$

# Answer

$$\boxed{V_B = \frac{5 w L}{8}}$$
