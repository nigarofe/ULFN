# Proposition
1. Usando o método dos coeficientes indeterminados, quais são as soluções particulares para $y'' + y' + y = G(t)$ de acordo com a forma de $G(t)$?

2. Resolva as seguintes equações diferenciais:

a) $y'' + 4y = e^{3t}$

b) $y'' + y' - 2y = \sin(t)$

c) $y'' - 4y = te^{t} + \cos(2t)$


# Step-by-step
## Questão 1
### Método dos Coeficientes Indeterminados (versão 1)
$$y = y_h + y_p$$

**Soma**
$$ay'' + by' + cy = G_1(t) + G_2(t)$$
$$y_p = y_{p1} + y_{p2}$$

**Multiplicação**
$$ay'' + by' + cy = G(t)$$

| Se $G(t)$ for...                     | Exemplo de $G(t)$      | Forma do Palpite $y_p$                            |
| :----------------------------------- | :--------------------- | :------------------------------------------------ |
| **Polinômio $\cdot$ Exponencial**    | $e^{nt} \cdot t^2$     | $e^{nt}(At^2 + Bt + C)$                           |
| **Exponencial $\cdot$ Seno/Cosseno** | $e^{nt} \cdot \cos(t)$ | $e^{nt}(A\cos(t) + B\sin(t))$                     |
| **Polinômio $\cdot$ Seno**           | $t^2 \cdot \sin(t)$    | $(At^2 + Bt + C)\cos(t) + (Dt^2 + Et + F)\sin(t)$ |

### Método dos Coeficientes Indeterminados (versão 2)
$$ay'' + by' + cy = G(t)$$

**Caso 1**
$$G(t) = e^{kt} P(t)$$
$$\boxed{y_p(t) = e^{kt} Q(t)}$$ 

**Caso 2**

$$G(t) = e^{kt} P(t) \cos(mt) \quad \text{ou} \quad G(t) = e^{kt} P(t) \sin(mt)$$
$$\boxed{y_p(t) = e^{kt} Q(t) \cos(mt) + e^{kt} R(t) \sin(mt)}$$

**Observações**

1. O grau dos polinômios $P(t)$, $Q(t)$ e $R(t)$ é o mesmo.
2. Se algum termo de $y_p$ for solução da eq homogênea, multiplique $y_p$ por $t$ (ou $t^2$ se preciso)

## Questão 2a
$$y'' + 4y = e^{3t}$$
$$r^2 + 4 = 0 \Rightarrow r = \pm \sqrt{-4} = 0 \pm 2i \implies \alpha = 0, \beta = 2$$
$$y_H = e^{\alpha t} (c_1 \cos(\beta t) + c_2 \sin(\beta t))$$
$$y_H = e^{0t} \left(c_1 \cos \left(2t\right) + c_2 \sin \left(2t\right)\right)$$
$$y_H = c_1 \cos \left(2t\right) + c_2 \sin \left(2t\right)$$

---

$$y_P = A e^{3t} \quad y_P' = 3A e^{3t} \quad y_P'' = 9A e^{3t}$$
$$\left[9A e^{3t}\right] + 4\left[A e^{3t}\right] = e^{3t}$$
$$13A = 1 \Rightarrow A = \frac{1}{13} \Rightarrow y_P = \frac{1}{13} e^{3t}$$

---

$$\boxed{y = c_1 \cos(2t) + c_2 \sin(2t) + \frac{1}{13} e^{3t}}$$

## Questão 2b
$$y'' + y' - 2y = \sin \left(t\right)$$
$$r^2 + r - 2 = 0$$
$$\Delta = b^2 - 4ac = 1^2 - 4 \cdot 1 \cdot \left(-2\right) = 9$$
$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-1 \pm \sqrt{9}}{2 \cdot 1} = \frac{-1 \pm 3}{2} = 1, -2$$
$$y_H = c_1 e^t + c_2 e^{-2t}$$

---

$$y_P = A \cos \left(t\right) + B \sin \left(t\right)$$
$$y_P' = -A \sin \left(t\right) + B \cos \left(t\right)$$
$$y_P'' = -A \cos \left(t\right) - B \sin \left(t\right)$$
$$\left[-A \cos \left(t\right) - B \sin \left(t\right)\right] + \left[-A \sin + B \cos \left(t\right)\right] - 2\left[A \cos \left(t\right) + B \sin \left(t\right)\right] = \sin \left(t\right)$$
$$\left(-A + B - 2A\right) \cos \left(t\right) + \left(-B - A - 2B\right) \sin \left(t\right) = \sin \left(t\right)$$

$$
\begin{cases}
-3A + B = 0 \\
-A - 3B = 1
\end{cases}
\implies
-10B = -3 \implies B = -\frac{3}{10} \quad A = -\frac{1}{10}
$$
$$y_P = -\frac{1}{10} \cos \left(t\right) - \frac{3}{10} \sin \left(t\right)$$

---

$$\boxed{y = c_1 e^t + c_2 e^{-2t} - \frac{1}{10} \cos \left(t\right) - \frac{3}{10} \sin \left(t\right)}$$

## Questão 2c
$$y'' - 4y = te^t + \cos(2t)$$

$$y = y_H + y_P$$
$$y_P = y_{P1} + y_{P2}$$

---

$$n^2 - 4 = 0$$
$$n^2 = +4$$
$$n = \pm \sqrt{4} = \pm 2$$
$$y_H = c_1 e^{2t} + c_2 e^{-2t}$$

---

$$y_{P1} = (At + B) e^t = Ae^t t + Be^t$$
$$y'_{P1} = A [ (e^t)(1) + (e^t)(t) ] + Be^t = Ae^t + Ae^t t + Be^t$$
$$y''_{P1} = Ae^t + Ae^t + Ae^t t + Be^t = (2A + At + B) e^t$$
$$[ (2A + At + B) e^t ] - 4 [ (At + B) e^t ] = te^t$$
$$(2A + At + B - 4At - 4B) e^t = te^t$$
$$-3At + 2A - 3B = 1t + 0$$
$$
\begin{cases}
-3A = 1 \\
2A - 3B = 0
\end{cases}
\qquad A = -\frac{1}{3}
\qquad B = \frac{2}{3} A = -\frac{2}{9}$$
$$y_{P1} = \left( -\frac{1}{3} t - \frac{2}{9} \right) e^t$$

---

$$y_{P2} = A \cos(2t) + B \sin(2t)$$
$$y'_{P2} = -2A \sin(2t) + 2B \cos(2t)$$
$$y''_{P2} = -4A \cos(2t) - 4B \sin(2t)$$
$$[ -4A \cos(2t) - 4B \sin(2t) ] - 4 [ A \cos(2t) + B \sin(2t) ] = \cos(2t)$$
$$[ -4A - 4A ]\cos(2t) + [ -4B - 4B ]\sin(2t) = [1]\cos(2t) + [0]\sin(2t)$$
$$
\begin{cases}
-8A = 1 \\
-8B = 0
\end{cases}
\qquad
A = -\frac{1}{8} 
\qquad B = 0
$$

$$y_{P2} = -\frac{1}{8} \cos(2t)$$

> Observação: Como não há $y'$ na EDO, $y_{P2}$ poderia ser apenas $A \cos(2t)$, já que os termos de $\sin(2t)$ não aparecem na substituição da EDO.

---

$$
\boxed{
y = c_1 e^{2t} + c_2 e^{-2t} + \left(-\frac{1}{3} t - \frac{2}{9}\right) e^t + \left(- \frac{1}{8} \cos(2t)\right)
}
$$
