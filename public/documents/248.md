# Proposition
Encontre a solução geral da seguinte EDO
$$u'' + \omega_0^2 u = \cos(\omega t)$$
onde $\omega, \omega_0 \in \mathbb{R}$

> Obs.: Perguntar para o professor se em problemas como esse, deve-se considerar ressonância ou não, ou seja, se $\omega = \omega_0$ ou $\omega \neq \omega_0$

# Step-by-step
## Caso $\omega \neq \omega_0$ (Sem ressonância)
$$u'' + \omega_0^2 u = \cos(\omega t)$$

$$r^2 + \omega_0^2 = 0$$

$$r^2 = -\omega_0^2$$

$$r = \pm \sqrt{-\omega_0^2} = 0 \pm i \omega_0 \to \alpha = 0, \beta = \omega_0$$

$$u_H = e^{\alpha t} (c_1 \cos(\beta t) + c_2 \sin(\beta t))$$

$$u_H = c_1 \cos(\omega_0 t) + c_2 \sin(\omega_0 t)$$

---

$$u_P = A \cos(\omega t) + B \sin(\omega t)$$

$$u_P' = -A \omega \sin(\omega t) + B \omega \cos(\omega t)$$

$$u_P'' = -A \omega^2 \cos(\omega t) - B \omega^2 \sin(\omega t)$$

$$[-A \omega^2 \cos(\omega t) - B \omega^2 \sin(\omega t)] + \omega_0^2 [A \cos(\omega t) + B \sin(\omega t)]$$
$$= \cos(\omega t)$$

$$\cos(\omega t) [-A \omega^2 + A \omega_0^2] + \sin(\omega t) [-B \omega^2 + B \omega_0^2] = \cos(\omega t)$$

$$
\begin{cases}
 -A \omega^2 + A \omega_0^2 = 1 \\
 -B \omega^2 + B \omega_0^2 = 0
\end{cases}
$$

$$A(-\omega^2 + \omega_0^2) = 1$$

$$A = \frac{1}{\omega_0^2 - \omega^2}$$

$$u_P = \frac{1}{\omega_0^2 - \omega^2} \cos(\omega t)$$

---

$$\boxed{u = c_1 \cos(\omega_0 t) + c_2 \sin(\omega_0 t) + \frac{1}{\omega_0^2 - \omega^2} \cos(\omega t)}$$

## Caso $\omega = \omega_0$ (Com ressonância)

> The solution below is by Gemini

$$u'' + \omega_0^2 u = \cos(\omega_0 t)$$

$$u_H = c_1 \cos(\omega_0 t) + c_2 \sin(\omega_0 t)$$

$$u_P = t(A \cos(\omega_0 t) + B \sin(\omega_0 t))$$

$$u_P' = [A \cos(\omega_0 t) + B \sin(\omega_0 t)] + t[-A \omega_0 \sin(\omega_0 t) + B \omega_0 \cos(\omega_0 t)]$$

$$u_P'' = [-A \omega_0 \sin(\omega_0 t) + B \omega_0 \cos(\omega_0 t)] + [-A \omega_0 \sin(\omega_0 t) + B \omega_0 \cos(\omega_0 t)] + t[-A \omega_0^2 \cos(\omega_0 t) - B \omega_0^2 \sin(\omega_0 t)]$$

$$u_P'' = -2A \omega_0 \sin(\omega_0 t) + 2B \omega_0 \cos(\omega_0 t) - \omega_0^2 t(A \cos(\omega_0 t) + B \sin(\omega_0 t))$$

Notice that the last term is exactly $-\omega_0^2 u_P$. So we can write:
$$u_P'' = -2A \omega_0 \sin(\omega_0 t) + 2B \omega_0 \cos(\omega_0 t) - \omega_0^2 u_P$$

**3. Substitute back into the original ODE**
Substitute $u_P$ and $u_P''$ into $u'' + \omega_0^2 u = \cos(\omega_0 t)$:

$$[-2A \omega_0 \sin(\omega_0 t) + 2B \omega_0 \cos(\omega_0 t) - \omega_0^2 u_P] + \omega_0^2 u_P = \cos(\omega_0 t)$$

The $\omega_0^2 u_P$ terms beautifully cancel out, leaving:
$$-2A \omega_0 \sin(\omega_0 t) + 2B \omega_0 \cos(\omega_0 t) = \cos(\omega_0 t)$$

**4. Match the coefficients**
Now we set up our system of equations by matching the sine and cosine terms on both sides:

*   **For sine terms:** $-2A \omega_0 = 0 \implies A = 0$
*   **For cosine terms:** $2B \omega_0 = 1 \implies B = \frac{1}{2\omega_0}$

**5. Write the final general solution**
Plug $A$ and $B$ back into our guess for $u_P$:
$$u_P = t \left(0 \cdot \cos(\omega_0 t) + \frac{1}{2\omega_0} \sin(\omega_0 t)\right) = \frac{t}{2\omega_0} \sin(\omega_0 t)$$


$$\boxed{u(t) = c_1 \cos(\omega_0 t) + c_2 \sin(\omega_0 t) + \frac{t}{2\omega_0} \sin(\omega_0 t)}$$

# Answer
## Caso $\omega \neq \omega_0$ (Sem ressonância)
$$\boxed{u = c_1 \cos(\omega_0 t) + c_2 \sin(\omega_0 t) + \frac{1}{\omega_0^2 - \omega^2} \cos(\omega t)}$$

## Caso $\omega = \omega_0$ (Com ressonância)
$$\boxed{u(t) = c_1 \cos(\omega_0 t) + c_2 \sin(\omega_0 t) + \frac{t}{2\omega_0} \sin(\omega_0 t)}$$
