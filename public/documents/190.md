# Proposition
Solve the following differential equations:

a) $y'' + 3y' + 2y = 4x$
b) $y'' + 4y' + 3y = 3x + 10$
c) $y'' + 5y' + 6y = 12x - 2$
d) $y'' + 6y' + 5y = 5x + 16$

Solved items:

- a
- b
- c
- d
- a

# Step-by-step
## a
$$y'' + 3y' + 2y = 4x$$

---

$$r^2 + 3r + 2 = 0$$
$$\Delta = b^2 - 4ac = 9 - 4(1)(2) = 9 - 8 = 1$$
$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-3 \pm \sqrt{1}}{2(1)} = \frac{-3 \pm 1}{2} = -1, -2$$
$$y_H = c_1e^{r_1x} + c_2e^{r_2x}$$
$$y_H = c_1e^{-x} + c_2e^{-2x}$$

---

$$y_P = Ax + B \implies y_P' = A \implies y_P'' = 0$$
$$(0) + 3(A) + 2(Ax + B) = 4x$$
$$3A + 2Ax + 2B = 4x \implies A = 2$$
$$6 + 4x + 2B = 4x \implies B = -3$$

$$y_P = 2x - 3$$

---

$$y = y_H + y_P = c_1e^{-x} + c_2e^{-2x} + 2x - 3$$

# Answer
a) $y = c_1e^{-x} + c_2e^{-2x} + 2x - 3$
b) $y = c_1e^{-x} + c_2e^{-3x} + x + 2$
c) $y = c_1e^{-2x} + c_2e^{-3x} + 2x - 2$
d) $y = c_1e^{-x} + c_2e^{-5x} + x + 2$
