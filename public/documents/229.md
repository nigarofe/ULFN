# Proposition
Solve the following differential equations:

a) $t^2 y' + 2ty - y^3 = 0 \qquad t > 0$

## Surprise test
Solve, if possible:

b) $ty' + 2y = t^2 - t + 1 \qquad y(1) = \frac{1}{2} \qquad t>0$
c) $(e^x \sin(y) + 3y) - (3x - e^x \sin(y)) y' = 0$

# Step-by-step
$$y' + p(t) y = q(t) y^n \quad v = y^{1-n}$$

## a
$$t^2 y' + 2ty - y^3 = 0 \quad t > 0$$
$$y' + \frac{2}{t} y = t^{-2} y^3$$

---

$$y' + p(t) y = q(t) y^n \quad v = y^{1-n}$$
$$p(t) = \frac{2}{t} \quad q(t) = t^{-2} \quad n = 3$$
$$v = y^{1-3} = y^{-2} = \left[ y(t) \right]^{-2}$$
$$v = f(y(t)) = \left[ y(t) \right]^{-2}$$
$$v' = f'(y(t)) \cdot y'(t) = -2 \left[ y(t) \right]^{-3} \cdot y'(t)$$
$$v' = -2 y^{-3} y'$$
$$y' = -\frac{v'}{2y^{-3}} = -\frac{1}{2} v' y^3$$

---

$$-\frac{1}{2} v' y^3 + \frac{2}{t} y = t^{-2} y^3$$
$$-\frac{1}{2} v' \frac{y^3}{y^3} + \frac{2}{t} \frac{y}{y^3} = t^{-2} \frac{y^3}{y^3}$$
$$-\frac{1}{2} v' + \frac{2}{ty^2} = t^{-2}$$
$$v' - \frac{4}{t} y^{-2} = -2t^{-2}$$
$$v' - \frac{4}{t} v = -2t^{-2}$$

---

$$\mu = e^{\int p dt} = e^{\int -\frac{4}{t} dt} = t^{-4}$$
$$v = \frac{1}{\mu} \left( \int \mu q dt + c \right) = \frac{1}{t^{-4}} \left( \int \left( t^{-4} \right) \left( -2t^{-2} \right) dt + c \right) = t^4 \left( -2 \int t^{-6} dt + c \right)$$
$$v = t^4 \left[ -2 \cdot \left( \frac{t^{-5}}{-5} \right) + c \right] = t^4 \left( \frac{2}{5} t^{-5} + c \right)$$
$$v = \frac{2}{5} t^{-1} + ct^4$$

---

$$v = y^{-2} = \frac{1}{y^2}$$
$$\frac{1}{y^2} = \frac{2}{5} t^{-1} + ct^4$$
$$\frac{1}{y^2} = \frac{2}{5t} + ct^4$$
$$\frac{1}{y^2} = \frac{2 + 5ct^5}{5t}$$
$$y^2 = \frac{5t}{2 + 5ct^5}$$
$$\boxed{y = \pm \sqrt{\frac{5t}{2 + 5ct^5}}}$$

## b
$$t y' + 2y = t^2 - t + 1 \qquad y(1) = \frac{1}{2} \qquad t > 0$$

$$y' + \frac{2}{t} y = t - 1 + \frac{1}{t}$$

$$\mu = e^{\int \frac{2}{t} dt} = t^2$$

$$y = \frac{1}{\mu} \left( \int \mu q dt + c \right) = \frac{1}{t^2} \left( \int (t^2) \left( t - 1 + \frac{1}{t} \right) dt + c \right)$$

$$y = \frac{1}{t^2} \left( \int (t^3 - t^2 + t) dt + c \right) = \frac{1}{t^2} \left( \frac{t^4}{4} - \frac{t^3}{3} + \frac{t^2}{2} + c \right)$$

$$y = \frac{t^2}{4} - \frac{t}{3} + \frac{1}{2} + c t^{-2}$$

$$y(1) = \frac{1}{4} - \frac{1}{3} + \frac{1}{2} + c = \frac{3 - 4 + 6}{12} + c = \frac{5}{12} + c = \frac{1}{2}$$

$$c = \frac{1}{2} - \frac{5}{12} = \frac{6 - 5}{12} = \frac{1}{12}$$

$$y = \frac{t^2}{4} - \frac{t}{3} + \frac{1}{2} + \frac{1}{12} t^{-2}$$

$$\boxed{y = \frac{t^2}{4} - \frac{t}{3} + \frac{1}{2} + \frac{1}{12t^2}}$$

## c
$$(e^x \sin(y) + 3y) - (3x - e^x \sin(y)) y' = 0$$

$$M = e^x \sin(y) + 3y \qquad N = -3x + e^x \sin(y)$$

$$M_y = e^x \cos(y) + 3 \qquad N_x = e^x \sin(y) - 3$$

$$\frac{M_y - N_x}{N} = \frac{(e^x \cos(y) + 3) - (e^x \sin(y) - 3)}{e^x \sin(y) - 3x} = \frac{e^x \cos(y) - e^x \sin(y) + 6}{e^x \sin(y) - 3x}$$

$$\frac{N_x - M_y}{M} = \frac{(e^x \sin(y) - 3) - (e^x \cos(y) + 3)}{e^x \sin(y) + 3y} = \frac{e^x \sin(y) - e^x \cos(y) - 6}{e^x \sin(y) + 3y}$$

Como $M_y \neq N_x$, a equação não é exata. Entretanto, podemos tentar encontrar um fator integrante $\mu(x)$ ou $\mu(y)$ tal que a equação se torne exata. Para isso, verificamos se $\frac{M_y - N_x}{N}$ é função apenas de $x$ ou se $\frac{N_x - M_y}{M}$ é função apenas de $y$, respectivamente. Como a equação não se encaixa em nenhuma dessas cenários, a equação não possui solução analítica por métodos elementares.


# Answer
## a
$$\boxed{y = \pm \sqrt{\frac{5t}{2 + 5ct^5}}}$$

## b
$$\boxed{y = \frac{t^2}{4} - \frac{t}{3} + \frac{1}{2} + \frac{1}{12t^2}}$$

## c
Como $M_y \neq N_x$, a equação não é exata. Entretanto, podemos tentar encontrar um fator integrante $\mu(x)$ ou $\mu(y)$ tal que a equação se torne exata. Para isso, verificamos se $\frac{M_y - N_x}{N}$ é função apenas de $x$ ou se $\frac{N_x - M_y}{M}$ é função apenas de $y$. Como a equação não se encaixa em nenhuma dessas cenários, a equação não possui solução analítica por métodos elementares.
