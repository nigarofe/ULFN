# Proposition
Solve the following differential equations:

a) $y'' + 3y' + 2y = 6e^x$
b) $y'' + 4y' + 3y = 30e^{2x}$
c) $y'' + 5y' + 6y = 36e^x$
d) $y'' + 2y' + y = 8e^x$
e) $y'' + 6y' + 8y = 70e^{3x}$
f) $y'' + 6y' + 5y = 36e^x$

Solved items: 

- a, b
- c
- d
- e
- f
- a

# Step-by-step
## b
$$y'' + 4y' + 3y = 30e^{2x}$$

---

$$r^2 + 4r + 3 = 0$$
$$\Delta = b^2 - 4ac = 16 - 4(1)(3) = 16 - 12 = 4$$
$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-4 \pm \sqrt{4}}{2(1)} = \frac{-4 \pm 2}{2} = -1, -3$$
$$y_H = c_1e^{r_1x} + c_2e^{r_2x}$$
$$y_H = c_1e^{-x} + c_2e^{-3x}$$

---

$$y_P = Ae^{2x} \implies y_P' = 2Ae^{2x} \implies y_P'' = 4Ae^{2x}$$
$$(4Ae^{2x}) + 4(2Ae^{2x}) + 3(Ae^{2x}) = 15Ae^{2x}$$
$$15A = 30 \implies A = 2$$
$$y_P = 2e^{2x}$$

---

$$y = y_H + y_P = c_1e^{-x} + c_2e^{-3x} + 2e^{2x}$$

# Answer
a) $y = c_1e^{-x} + c_2e^{-2x} + e^x$
b) $y = c_1e^{-x} + c_2e^{-3x} + 2e^{2x}$
c) $y = c_1e^{-2x} + c_2e^{-3x} + 3e^x$
d) $y = c_1e^{-x} + c_2xe^{-x} + 2e^x$
e) $y = c_1e^{-2x} + c_2e^{-4x} + 2e^{3x}$
f) $y = c_1e^{-x} + c_2e^{-5x} + 3e^x$
