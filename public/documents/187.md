# Proposition
Solve the following differential equations:

a) $y'' + 4y' + 5y = 0$, where $y(0) = 1$ and $y'(0) = -1$.
b) $y'' + 4y = 0$, where $y(0) = 1$ and $y'(0) = 1$

Solved items: 

- a, b
- a
- b
- a
- b
- a

# Step-by-step
## a - Fastest speed method 
**Attention:** this method only works when the given initial conditions are $y(0)$ and $y'(0)$

$$
y'' + 4y' + 5y = 0
\quad
y(0) = 1
\quad
y'(0) = -1
$$
$$r^2 + 4r + 5 = 0$$
$$\Delta = b^2 - 4ac = 4^2 - 4(1)(5) = 16 - 20 = -4$$
$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-4 \pm \sqrt{-4}}{2(1)} = \frac{-4 \pm 2i}{2} = -2 \pm 1i$$
$$y(x) = e^{mx} (c_1 \sin nx + c_2 \cos nx)$$
$$y(x) = e^{-2x} (c_1 \sin x + c_2 \cos x)$$
$$y(0) = c_2 = 1 \implies c_2 = 1$$
$$y(x) = e^{-2x} (c_1 \sin x + \cos x)$$
$$y'(0) = m (y(0)) + n c_1$$
$$y'(0) = -2 (y(0)) + c_1 = -2 + c_1 = -1 \implies c_1 = 1$$
$$y(x) = e^{-2x} ( \sin x + \cos x )$$

## a - Medium speed method
$$
y'' + 4y' + 5y = 0
\quad
y(0) = 1
\quad
y'(0) = -1
$$
$$r^2 + 4r + 5 = 0$$
$$\Delta = b^2 - 4ac = 4^2 - 4(1)(5) = 16 - 20 = -4$$
$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-4 \pm \sqrt{-4}}{2(1)} = \frac{-4 \pm 2i}{2} = -2 \pm 1i$$
$$y(x) = e^{mx} (c_1 \sin nx + c_2 \cos nx)$$
$$y(x) = e^{-2x} (c_1 \sin x + c_2 \cos x)$$
$$y(0) = c_2 = 1 \implies c_2 = 1$$
$$y(x) = e^{-2x} (c_1 \sin x + \cos x)$$
$$y'(x) = ...$$

---

$$\frac{d}{dx} e^{-2x} (c_1 \sin x + \cos x) = ...$$
$$\frac{d}{dx} fg = fg' + f'g$$
$$f = e^{-2x} \quad f'=-2e^{-2x}$$
$$g = c_1 \sin x + \cos x \quad g' = c_1 \cos x - \sin x$$
$$\frac{d}{dx} e^{-2x} (c_1 \sin x + \cos x) = e^{-2x} (c_1 \cos x - \sin x) - 2e^{-2x} (c_1 \sin x + \cos x)$$

---

$$y'(0) = c_1 - 2 = -1 \implies c_1 = 1$$
$$y(x) = e^{-2x} ( \sin x + \cos x )$$


## a - Slowest speed method
$$
y'' + 4y' + 5y = 0
\quad
y(0) = 1
\quad
y'(0) = -1
$$
$$r^2 + 4r + 5 = 0$$
$$\Delta = b^2 - 4ac = 4^2 - 4(1)(5) = 16 - 20 = -4$$
$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-4 \pm \sqrt{-4}}{2(1)} = \frac{-4 \pm 2i}{2} = -2 \pm 1i$$
$$y=c_1 e^{mx} \sin nx + c_2 e^{mx} \cos nx$$
$$y = c_1 e^{-2x} \sin x + c_2 e^{-2x} \cos x$$
$$y' = ...$$

---

$$\frac{d}{dx} e^{-2x} \sin x = ...$$
$$\frac{d}{dx} fg = fg' + f'g$$
$$f = e^{-2x} \quad f'=-2e^{-2x}$$
$$g = \sin x \quad g'=\cos x$$
$$\frac{d}{dx} e^{-2x} \sin x = e^{-2x} \cos x - 2e^{-2x} \sin x $$

---

$$\frac{d}{dx} e^{-2x} \cos x = ...$$
$$\frac{d}{dx} fg = fg' + f'g$$
$$f = e^{-2x} \quad f'=-2e^{-2x}$$
$$g = \cos x \quad g'=-\sin x$$
$$\frac{d}{dx} e^{-2x} \cos x = -e^{-2x} \sin x - 2e^{-2x} \cos x$$

---

$$y' = c_1(e^{-2x} \cos x - 2 e^{-2x} \sin x) + c_2(-e^{-2x} \sin x - 2 e^{-2x} \cos x)$$

---

$$y(0) = c_2 = 1 \implies c_2 = 1$$
$$y'(0) = c_1 - 2c_2 = -1 \implies c_1 = 1$$
$$y(x) = e^{-2x} \sin x + e^{-2x} \cos x$$


# Answer
a) $y(x) = e^{-2x}( \cos x + \sin x )$
b) $y(x) = \cos 2x + \frac{1}{2} \sin 2x$
