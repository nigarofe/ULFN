# Proposition
Find the Laplace transform of the following functions:

a) $y^{(4)}$
b) $y^{(5)}$

Solved items:
- a
- b
- a
- a

# Step-by-step

$$\mathcal{L}\{y^{(n)}\} = s^n \mathcal{L}\{y\} - \sum_{k=1}^n s^{k-1} y^{(n-k)}(0)$$

# Answers

a) $\mathcal{L}\{y^{(4)}\} = s^4 \mathcal{L}\{y\} - y'''(0) - sy''(0) - s^2y'(0) - s^3y(0)$

b) $\mathcal{L}\{y^{(5)}\} = s^5 \mathcal{L}\{y\} - y^{(4)}(0) - sy'''(0) - s^2y''(0) - s^3y'(0) - s^4y(0)$
