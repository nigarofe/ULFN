# Proposition
Solve for y in these differential equations:

a) $\frac{dy}{dx} = 4y - 8$, where $y(0) = 5$

b) $\frac{dy}{dx} = 3y - 9$, where $y(0) = 9$

c) $\frac{dy}{dx} = 9y - 18$, where $y(0) = 5$

d) $\frac{dy}{dx} = 4y - 20$, where $y(0) = 16$

Solved items:
- a, b, c, d
- a, b
- c, d
- a

# Step-by-step

# Answers
a) $y = 2 + 3e^{4x}$
b) $y = 3 + 6e^{3x}$
c) $y = 2 + 3e^{9x}$
d) $y = 5 + 11e^{4x}$
