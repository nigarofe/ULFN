# Proposition
Considere a seguinte EDO
$$ y'' - 2y' + y = 0 $$

a) Verifique se $y_1(t) = e^t$ e $y_2(t) = te^t$ são soluções da EDO acima.

b) Elas formam um conjunto fundamental de soluções? **Justifique** matematicamente sua resposta.

# Step-by-step
$$y'' - 2y' + y = 0$$

$$y_1 = e^t \quad y_1' = e^t \quad y_1'' = e^t$$

$$(e^t) - 2(e^t) + (e^t) = 0 \quad \checkmark$$

$$y_2 = te^t \quad y_2' = (t)(e^t) + (1)(e^t) = te^t + e^t$$

$$y_2'' = te^t + e^t + e^t$$

$$(te^t + 2e^t) - 2(te^t + e^t) + (te^t) = 0 \quad \checkmark$$

---

$$W(y_1, y_2)(t) = \text{det} \begin{bmatrix} y_1 & y_2 \cr y_1' & y_2' \end{bmatrix} = y_1 y_2' - y_2 y_1'$$

$$W = (e^t)(te^t + e^t) - (te^t)(e^t) = te^{2t} + e^{2t} - te^{2t} = e^{2t}$$

Como $W \neq 0$, para todo $t$, $y_1$ e $y_2$ formam um conjunto fundamental de soluções.

# Answer
a) Sim, $y_1(t) = e^t$ e $y_2(t) = te^t$ são soluções da EDO dada.

b) Sim, elas formam um conjunto fundamental de soluções, pois o Wronskiano $W(y_1, y_2)(t) = e^{2t} \neq 0$ para todo $t$.
