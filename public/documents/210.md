# Proposition
Solve the following differential equations using the Laplace transform:

a) $y'' + 3y' + 2y = 0$, where $y(0) = 2$ and $y'(0) = -3$
b) $y'' + 5y' + 4y = 0$, where $y(0) = 2$ and $y'(0) = -5$
c) $y'' + 5y' + 6y = 0$, where $y(0) = 2$ and $y'(0) = -5$
d) $y'' + 4y' + 3y = 0$, where $y(0) = 3$ and $y'(0) = -5$
e) $y'' + 6y' + 5y = 0$, where $y(0) = 5$ and $y'(0) = -9$
f) $y'' + 6y' + 8y = 0$, where $y(0) = 4$ and $y'(0) = -14$

Solved items:

- a
- b
- c
- d
- e

# Step-by-step
## a
$$y'' + 3y' + 2y = 0 \quad \quad y(0) = 2 \quad \quad y'(0) = -3$$

$$\mathcal{L}\{y''\} + 3 \mathcal{L}\{y'\} + 2 \mathcal{L}\{y\} = 0$$

$$\mathcal{L}\{y^{(n)}\} = s^n \mathcal{L}\{y\} - \sum_{k=1}^n s^{k-1} y^{(n-k)}(0)$$

$$\mathcal{L}\{y''\} = s^2 \mathcal{L}\{y\} - y'(0) - s y(0)$$

$$\mathcal{L}\{y'\} = s \mathcal{L}\{y\} - y(0)$$

$$[s^2 \mathcal{L}\{y\} - y'(0) - s y(0)] + 3 [s \mathcal{L}\{y\} - y(0)] + 2 \mathcal{L}\{y\} = 0$$

$$[s^2 + 3s + 2] \mathcal{L}\{y\} + [-1] y'(0) + [-s - 3] y(0) = 0$$

$$[s^2 + 3s + 2] \mathcal{L}\{y\} + [-1] (-3) + [-s - 3] (2) = 0$$

$$\mathcal{L}\{y\} = \frac{2s + 3}{s^2 + 3s + 2} = \frac{2s + 3}{(s+1)(s+2)} = \frac{a}{s+1} + \frac{b}{s+2} = \frac{a(s+2) + b(s+1)}{(s+1)(s+2)}$$

$$2s + 3 = a(s+2) + b(s+1) \rightarrow a = 1, b = 1$$

$$\mathcal{L}\{y\} = \frac{1}{s+1} + \frac{1}{s+2}$$

$$\mathcal{L}\{e^{at}\} = \frac{1}{s-a} \quad s > a$$

$$\mathcal{L}^{-1}\left\{\frac{1}{s+1}\right\} = e^{-t}$$

$$\mathcal{L}^{-1}\left\{\frac{1}{s+2}\right\} = e^{-2t}$$

$$y(t) = e^{-t} + e^{-2t}$$

## c
$$y'' + 5y' + 6y = 0 \quad y(0) = 2 \quad y'(0) = -5$$

$$\mathcal{L}\{y^{(n)}\} = s^n \mathcal{L}\{y\} - \sum_{k=1}^n s^{k-1} y^{(n-k)}(0)$$

$$\mathcal{L}\{y''\} = s^2 \mathcal{L}\{y\} - y'(0) - s y(0)$$

$$\mathcal{L}\{y'\} = s \mathcal{L}\{y\} - y(0)$$

$$(s^2 \mathcal{L}\{y\} - y'(0) - s y(0)) + 5(s \mathcal{L}\{y\} - y(0)) + 6 \mathcal{L}\{y\} = 0$$

$$\mathcal{L}\{y\} [s^2 + 5s + 6] + y'(0)[-1] + y(0)[-s - 5] = 0$$

$$\mathcal{L}\{y\} [s^2 + 5s + 6] + 5 + 2(-s - 5) = 0$$

$$\mathcal{L}\{y\} = \frac{2s + 5}{s^2 + 5s + 6} = \frac{2s + 5}{(s+3)(s+2)} = \frac{a}{(s+3)} + \frac{b}{(s+2)} = \frac{a(s+2) + b(s+3)}{(s+3)(s+2)}$$

$$2s + 5 = a(s+2) + b(s+3)$$

$$s = -3 \rightarrow 2(-3) + 5 = a(-3+2) + b(-3+3) \rightarrow -1 = -a \rightarrow a = 1$$

$$s = -2 \rightarrow 2(-2) + 5 = a(-2+2) + b(-2+3) \rightarrow 1 = b$$

$$\mathcal{L}\{y\} = \frac{1}{s+3} + \frac{1}{s+2}$$

$$\mathcal{L}\{e^{at}\} = \frac{1}{s-a} \quad s > a$$

$$\mathcal{L}^{-1}\left\{\frac{1}{s+3}\right\} = e^{-3t}$$

$$\mathcal{L}^{-1}\left\{\frac{1}{s+2}\right\} = e^{-2t}$$

$$y(t) = e^{-3t} + e^{-2t}$$


# Answers
a) $y = e^{-t} + e^{-2t}$
b) $y = e^{-t} + e^{-4t}$
c) $y = e^{-2t} + e^{-3t}$
d) $y = 2e^{-t} + e^{-3t}$
e) $y = 4e^{-t} + e^{-5t}$
f) $y = e^{-2t} + 3e^{-4t}$
