# Proposition
a) $$y'' + y = \sin(2t) + \sin(3t) + \cdots$$

b) $$y'' + y = \sin(t) + \sin(2t) + \sin(3t) + \cdots$$

# Notes


# Step-by-step
## a
$$y'' + y = \sin(2t) + \sin(3t) + \cdots$$

$$r^2 + 1 = 0$$

$$r = \pm \sqrt{-1} = 0 \pm 1i \Rightarrow \alpha = 0, \beta = 1$$

$$y_H = e^{\alpha t} (c_1 \cos(\beta t) + c_2 \sin(\beta t))$$

$$y_H = c_1 \cos(t) + c_2 \sin(t)$$

$$y_P = \sum_{n=2}^{\infty} (A_n \cos(nt) + B_n \sin(nt))$$

$$y'_P = \sum_{n=2}^{\infty} (-n A_n \sin(nt) + n B_n \cos(nt))$$

$$y''_P = \sum_{n=2}^{\infty} (-n^2 A_n \cos(nt) - n^2 B_n \sin(nt))$$

$$y''_P + y_P = \sum_{n=2}^{\infty} \sin(nt)$$

$$\sum_{n=2}^{\infty} (-n^2 A_n \cos(nt) - n^2 B_n \sin(nt)) + \sum_{n=2}^{\infty} (A_n \cos(nt) + B_n \sin(nt)) = \sum_{n=2}^{\infty} \sin(nt)$$

$$\sum_{n=2}^{\infty} (A_n(1 - n^2) \cos(nt) + B_n(1 - n^2) \sin(nt)) = \sum_{n=2}^{\infty} \sin(nt)$$

$$A_n(1 - n^2) = 0 \Rightarrow A_n = 0$$

$$B_n(1 - n^2) = 1 \Rightarrow B_n = \frac{1}{1 - n^2}$$

$$y_P = \sum_{n=2}^{\infty} \frac{1}{1 - n^2} \sin(nt)$$

$$y = y_H + y_P$$

$$y = c_1 \cos(t) + c_2 \sin(t) + \sum_{n=2}^{\infty} \frac{1}{1 - n^2} \sin(nt)$$

## b
$$y'' + y = \sin(t) + \sin(2t) + \sin(3t) + \cdots$$

$$r^2 + 1 = 0$$

$$r = \pm \sqrt{-1} = 0 \pm 1i \Rightarrow \alpha = 0, \beta = 1$$

$$y_H = e^{\alpha t} (c_1 \cos(\beta t) + c_2 \sin(\beta t))$$

$$y_H = c_1 \cos(t) + c_2 \sin(t)$$

$$y_P = A_1 t \cos(t) + B_1 t \sin(t) + \sum_{n=2}^{\infty} (A_n \cos(nt) + B_n \sin(nt))$$

$$y'_P = A_1 (-t \sin(t) + 1 \cos(t)) + B_1 (t \cos(t) + 1 \sin(t)) + \sum_{n=2}^{\infty} (-n A_n \sin(nt) + n B_n \cos(nt))$$

$$y'_P = -A_1 t \sin(t) + A_1 \cos(t) + B_1 t \cos(t) + B_1 \sin(t) + \sum_{n=2}^{\infty} (-n A_n \sin(nt) + n B_n \cos(nt))$$

$$y''_P = -A_1 (t \cos(t) + 1 \cdot \sin(t)) - A_1 \sin(t) + B_1 (-t \sin(t) + 1 \cdot \cos(t)) + B_1 \cos(t) + \sum_{n=2}^{\infty} (-n^2 A_n \cos(nt) - n^2 B_n \sin(nt))$$

$$y''_P = -A_1 t \cos(t) - 2A_1 \sin(t) - B_1 t \sin(t) + 2B_1 \cos(t) + \sum_{n=2}^{\infty} (-n^2 A_n \cos(nt) - n^2 B_n \sin(nt))$$

$$y''_P + y_P = \sin(t) + \sum_{n=2}^{\infty} \sin(nt)$$

$$-A_1 t \cos(t) - 2A_1 \sin(t) - B_1 t \sin(t) + 2B_1 \cos(t) + \sum_{n=2}^{\infty} (-n^2 A_n \cos(nt) - n^2 B_n \sin(nt)) + A_1 t \cos(t) + B_1 t \sin(t) + \sum_{n=2}^{\infty} (A_n \cos(nt) + B_n \sin(nt)) = \sin(t) + \sum_{n=2}^{\infty} \sin(nt)$$

$$2B_1 \cos(t) - 2A_1 \sin(t) + \sum_{n=2}^{\infty} (A_n(1 - n^2) \cos(nt) + B_n(1 - n^2) \sin(nt)) = \sin(t) + \sum_{n=2}^{\infty} \sin(nt)$$

$$2B_1 = 0 \Rightarrow B_1 = 0$$

$$-2A_1 = 1 \Rightarrow A_1 = -\frac{1}{2}$$

$$A_n(1 - n^2) = 0 \Rightarrow A_n = 0$$

$$B_n(1 - n^2) = 1 \Rightarrow B_n = \frac{1}{1 - n^2}$$

$$y_P = -\frac{1}{2} t \cos(t) + \sum_{n=2}^{\infty} \frac{1}{1 - n^2} \sin(nt)$$

$$y = y_H + y_P$$

$$y = c_1 \cos(t) + c_2 \sin(t) - \frac{1}{2} t \cos(t) + \sum_{n=2}^{\infty} \frac{1}{1 - n^2} \sin(nt)$$
