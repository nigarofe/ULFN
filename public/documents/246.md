# Proposition
Resolva as seguintes equações diferenciais por meio de séries de potências:

a) $y'' + y = 0$

b) $y'' - 2xy' + y = 0 \qquad y(0) = 0 \qquad y'(0) = 1$

# Step-by-step

> See document 201 for a similar problem.

## a
$$y'' + y = 0$$

$$y = \sum_{n=0}^{\infty} a_n x^n = a_0 + a_1 x^1 + a_2 x^2 + a_3 x^3 + a_4 x^4 + \dots$$

$$y' = a_1 + 2 a_2 x^1 + 3 a_3 x^2 + 4 a_4 x^3 + \dots = \sum_{n=1}^{\infty} n a_n x^{n-1} = \sum_{n=0}^{\infty} (n+1) a_{n+1} x^n$$

$$y'' = 2 a_2 + 6 a_3 x + 12 a_4 x^2 + \dots = \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2} = \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} x^n$$

$$\left[ \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} x^n \right] + \left[ \sum_{n=0}^{\infty} a_n x^n \right] = 0$$

$$\left[ \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} + a_n \right] x^n = 0$$

$$(n+2)(n+1) a_{n+2} + a_n = 0$$

$$a_{n+2} = -\frac{a_n}{(n+2)(n+1)}$$

---

$$n = 0 \rightarrow a_2 = -\frac{a_0}{(2)(1)}$$

$$n = 2 \rightarrow a_4 = -\frac{a_2}{(4)(3)} = +\frac{a_0}{(4)(3)(2)(1)}$$

$$a_n = a_{2m} = (-1)^m \frac{a_0}{(2m)!} \qquad m = 0, 1, 2, 3, \dots$$

---

$$n = 1 \rightarrow a_3 = -\frac{a_1}{(3)(2)}$$

$$n = 3 \rightarrow a_5 = -\frac{a_3}{(5)(4)} = +\frac{a_1}{(5)(4)(3)(2)}$$

$$a_n = a_{2m+1} = (-1)^m \frac{a_1}{(2m+1)!} \qquad m = 0, 1, 2, 3, \dots$$

---

$$y = \sum_{n=0}^{\infty} a_n x^n = a_0 + a_1 x^1 + a_2 x^2 + a_3 x^3 + a_4 x^4 + \dots$$

$$y = \sum_{m=0}^{\infty} (-1)^m \frac{a_0}{(2m)!} x^{(2m)} + \sum_{m=0}^{\infty} (-1)^m \frac{a_1}{(2m+1)!} x^{(2m+1)}$$

$$y = a_0 \sum_{m=0}^{\infty} (-1)^m \frac{x^{(2m)}}{(2m)!} + a_1 \sum_{m=0}^{\infty} (-1)^m \frac{x^{(2m+1)}}{(2m+1)!}$$

$$\boxed{y = a_0 \cos(x) + a_1 \sin(x)}$$

## b
$$y'' - 2xy' + y = 0 \qquad y(0) = 0 \qquad y'(0) = 1$$

$$y = \sum_{n=0}^{\infty} a_n x^n = a_0 + a_1 x + a_2 x^2 + a_3 x^3 + a_4 x^4 + \dots$$

$$y' = a_1 + 2 a_2 x + 3 a_3 x^2 + 4 a_4 x^3 + \dots = \sum_{n=1}^{\infty} n a_n x^{n-1}$$

$$y'' = 2 a_2 + 6 a_3 x + 12 a_4 x^2 + \dots = \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2} = \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} x^n$$

---

$$y(0) = 0 \quad y'(0) = 1$$

$$y(0) = a_0 = 0 \implies a_0 = 0$$

$$y'(0) = a_1 = 1 \implies a_1 = 1$$

---

$$\left[ \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} x^n \right] - 2x \left[ \sum_{n=1}^{\infty} n a_n x^{n-1} \right] + \left[ \sum_{n=0}^{\infty} a_n x^n \right] = 0$$

---

$$-2x \left[ \sum_{n=1}^{\infty} n a_n x^{n-1} \right] = \sum_{n=1}^{\infty} -2 n a_n x^n$$

Como quando $n=0$, $-2 n a_n x^n = 0$, $\sum_{n=1}^{\infty} -2 n a_n x^n = \sum_{n=0}^{\infty} -2 n a_n x^n$

---

$$\left[ \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} - 2 n a_n + a_n \right] x^n = 0$$

Como a expressão acima é nula para qualquer $x$

$$(n+2)(n+1) a_{n+2} - 2 n a_n + a_n = 0$$

$$a_{n+2} = \frac{2 n a_n - a_n}{(n+2)(n+1)}$$

$$a_{n+2} = \frac{(2n - 1)}{(n+2)(n+1)} a_n$$

---

$$n=0 \to a_2 = \frac{(0-1)}{(2)(1)} a_0 = -\frac{1}{(2)(1)} a_0$$

$$n=2 \to a_4 = \frac{(4-1)}{(4)(3)} a_2 = + \frac{3}{(4)(3)} a_2 = -\frac{1 \cdot 3}{4!} a_0$$

$$n=4 \to a_6 = \frac{(8-1)}{(6)(5)} a_4 = + \frac{7}{(6)(5)} a_4 = -\frac{1 \cdot 3 \cdot 7}{6!} a_0$$

$$n=6 \to a_8 = \frac{(12-1)}{(8)(7)} a_6 = + \frac{11}{(8)(7)} a_6 = -\frac{1 \cdot 3 \cdot 7 \cdot 11}{8!} a_0$$

$$a_n = a_{2m} = + \frac{\prod_{k=1}^m (4k-5)}{(2m)!} a_0 \qquad m = 0, 1, 2, 3, \dots$$

---

$$n=1 \to a_3 = \frac{(2-1)}{(3)(2)} a_1 = + \frac{1}{(3)(2)} a_1$$

$$n=3 \to a_5 = \frac{(6-1)}{(5)(4)} a_3 = + \frac{5}{(5)(4)} a_3 = +\frac{1 \cdot 5}{5!} a_1$$

$$n=5 \to a_7 = \frac{(10-1)}{(7)(6)} a_5 = + \frac{9}{(7)(6)} a_5 = +\frac{1 \cdot 5 \cdot 9}{7!} a_1$$

$$n=7 \to a_9 = \frac{(14-1)}{(9)(8)} a_7 = + \frac{13}{(9)(8)} a_7 = +\frac{1 \cdot 5 \cdot 9 \cdot 13}{9!} a_1$$

$$a_n = a_{2m+1} = + \frac{\prod_{k=1}^m (4k-3)}{(2m+1)!} a_1 \qquad m = 0, 1, 2, 3, \dots$$

---

$$y = \left( a_0 + a_2 x^2 + a_4 x^4 + \dots \right) + \left( a_1 + a_3 x^3 + a_5 x^5 + \dots \right)$$

$$y = \sum_{m=0}^{\infty} \left[ + \frac{\prod_{k=1}^m (4k-5)}{(2m)!} a_0 x^{(2m)} + \frac{\prod_{k=1}^m (4k-3)}{(2m+1)!} a_1 x^{(2m+1)} \right]$$

Como $a_0 = 0$ e $a_1 = 1$

$$\boxed{y = \sum_{m=0}^{\infty} \frac{\prod_{k=1}^m (4k-3)}{(2m+1)!} x^{(2m+1)}}$$

---

**A convenção do produto vazio**

Um produto vazio é igual a 1. Isso acontece quando $m=0$, resultando em $\prod_{k=1}^0$.


# Answer
## a
$$\boxed{y = a_0 \cos(x) + a_1 \sin(x)}$$

## b
$$\boxed{y = \sum_{m=0}^{\infty} \frac{\prod_{k=1}^m (4k-3)}{(2m+1)!} x^{(2m+1)}}$$
