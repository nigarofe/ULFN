# Proposition

Solve the following differential equations:

a) $(4+t^2) y' + 2ty = 4t$

b) $y' + \frac{1}{2} y = \frac{1}{2} e^{\frac{t}{3}}$

c) $y' - 2y = 4 - t$

## Geral
$$y' + p(t)y = g(t)$$
$$y = \frac{1}{\mu(t)} \left(\int \mu(t) g(t) dt + c \right)$$
$$\mu(t) = e^{\int p(t) dt}$$

## a
$$(4 + t^2)y' + 2ty = 4t$$

$$(4 + t^2)\frac{d}{dt}y + 2ty = 4t$$

--- 

$$f(t) = 4 + t^2 \quad f'(t) = 2t$$

$$\frac{d}{dt} f(t)y(t) = f(t)y'(t) + f'(t)y(t)$$

---

$$\frac{d}{dt} (4 + t^2)y = 4t$$

$$\int \frac{d}{dt} (4 + t^2)y \, dt = \int 4t \, dt$$

$$(4 + t^2)y = 2t^2 + c$$

$$y = \frac{2t^2 + c}{4 + t^2}$$

## b
$$y' + \frac{1}{2} y = \frac{1}{2} e^{t/3}$$

$$y' + p(t) y = q(t)$$

$$\implies y = \frac{1}{\mu} \left(\int \mu q(t) dt + c\right) \quad \mu(t) = e^{\int p(t) dt}$$

$$p(t) = \frac{1}{2} \implies \mu(t) = e^{\int \frac{1}{2} dt} = e^{t/2}$$

$$y = \frac{1}{e^{t/2}} \left(\int e^{t/2} \cdot \frac{1}{2} e^{t/3} dt + c\right) = \frac{1}{e^{t/2}} \left(\frac{1}{2} \int e^{t/2 + t/3} dt + c\right)$$

$$= \frac{1}{e^{t/2}} \left(\frac{1}{2} \int e^{5t/6} dt + c\right)$$

---

$$\int e^{5t/6} dt \implies u = \frac{5t}{6} \implies \frac{du}{dt} = \frac{5}{6} \implies dt = \frac{6}{5} du$$

$$\int e^{5t/6} dt = \int e^u \cdot \frac{6}{5} du = \frac{6}{5} e^{5t/6}$$

---

$$= \frac{1}{e^{t/2}} \left(\frac{1}{2} \cdot \frac{6}{5} e^{5t/6} + c \right) = \frac{3}{5} \frac{e^{5t/6}}{e^{t/2}} + c e^{-t/2}$$

$$= \frac{3}{5} e^{\frac{5t-3t}{6}} + c e^{-t/2} = \frac{3}{5} e^{t/3} + c e^{-t/2}$$

## c
$$y' - 2y = 4 - t$$

$$y' + p(t)y = g(t) \qquad y = \frac{1}{\mu} \left(\int \mu g dt + c\right) \qquad \mu = e^{\int p dt}$$

$$\mu = e^{\int -2 dt} = e^{-2t}$$

$$y = \frac{1}{e^{-2t}} \left(\int e^{-2t} (4 - t) dt + c\right) = \frac{1}{e^{-2t}} \left(4 \int e^{-2t} dt - \int t e^{-2t} dt + c\right)$$

---

$$\int e^{-2t} dt$$

$$u = -2t \implies \frac{du}{dt} = -2 \implies dt = -\frac{1}{2} du$$

$$= \int e^u \left(-\frac{1}{2}\right) du = -\frac{1}{2} e^{-2t}$$

---

$$\int t e^{-2t} dt$$
$$\int uv' = uv - \int u'v$$

$$u = t \qquad u' = 1$$

$$v' = e^{-2t} \qquad v = -\frac{1}{2} e^{-2t}$$

$$\int t e^{-2t} dt = t\left(-\frac{1}{2} e^{-2t}\right) - \int (1)\left(-\frac{1}{2} e^{-2t}\right) dt$$

$$= -\frac{1}{2} t e^{-2t} + \frac{1}{2} \int e^{-2t} dt$$

$$= -\frac{1}{2} t e^{-2t} - \frac{1}{4} e^{-2t}$$

---

$$y = \frac{1}{e^{-2t}} \left(4\left(-\frac{1}{2} e^{-2t}\right) - \left(-\frac{1}{2} t e^{-2t} - \frac{1}{4} e^{-2t}\right) + c\right)$$

$$y = \frac{1}{e^{-2t}} \left(-2 e^{-2t} + \frac{1}{2} t e^{-2t} + \frac{1}{4} e^{-2t} + c\right)$$

$$y = -2 + \frac{1}{2} t + \frac{1}{4} + c e^{2t}$$
$$y = \frac{1}{2} t - \frac{7}{4} + c e^{2t}$$

# Answer
a) $y = \frac{2t^2 + c}{4 + t^2}$

b) $y = \frac{3}{5} e^{t/3} + c e^{-t/2}$

c) $y = \frac{1}{2} t - \frac{7}{4} + c e^{2t}$
