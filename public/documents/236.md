# Proposition
Solve the following differential equations:

a) $y'' - y = 0 \qquad y(0) = 2 \qquad y'(0) = -1$

b) $y'' + 5y' + 6y = 0 \qquad y(0) = 2 \qquad y'(0) = 3$

# Step-by-step
## a
$$y'' - y = 0 \quad y(0) = 2 \quad y'(0) = -1$$
$$r^2 - 1 = 0$$
$$r^2 = 1$$
$$r = \pm 1$$
$$y = c_1 e^x + c_2 e^{-x} \quad y(0) = c_1 + c_2 = 2$$
$$y' = c_1 e^x - c_2 e^{-x} \quad y'(0) = c_1 - c_2 = -1$$
$$c_1 = \frac{1}{2} \quad c_2 = \frac{3}{2}$$
$$y = \frac{1}{2} e^x + \frac{3}{2} e^{-x}$$

## b
$$y'' + 5y' + 6y = 0 \quad y(0) = 2 \quad y'(0) = 3$$
$$r^2 + 5r + 6 = 0$$
$$\Delta = b^2 - 4ac = 5^2 - 4 \cdot 1 \cdot 6 = 1$$
$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-5 \pm \sqrt{1}}{2 \cdot 1} = \frac{-5 \pm 1}{2} = -2, -3$$
$$y = c_1 e^{-2x} + c_2 e^{-3x} \quad y(0) = c_1 + c_2 = 2$$
$$y' = -2c_1 e^{-2x} - 3c_2 e^{-3x} \quad y'(0) = -2c_1 - 3c_2 = 3$$
$$c_2 = -7 \quad c_1 = 9$$
$$y = 9e^{-2x} - 7e^{-3x}$$

# Answer
$$\boxed{y = \frac{1}{2} e^x + \frac{3}{2} e^{-x}}$$
$$\boxed{y = 9e^{-2x} - 7e^{-3x}}$$
