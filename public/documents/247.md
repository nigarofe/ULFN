# Proposition
Resolva os seguintes PVI:

a) $y'' + 2y' + 2y = 0 \qquad y\left(\frac{\pi}{4}\right) = 2 \qquad y'\left(\frac{\pi}{4}\right) = -2$

b) $y'' - 2y' - 3y = 3te^{2t} \qquad y(0) = 1 \qquad y'(0) = 0$


# Step-by-step
## a
$$y'' + 2y' + 2y = 0 \quad y\left(\frac{\pi}{4}\right) = 2 \quad y'\left(\frac{\pi}{4}\right) = -2$$

$$r^2 + 2r + 2 = 0$$

$$\Delta = b^2 - 4ac = 2^2 - 4 \cdot 1 \cdot 2 = 4 - 8 = -4$$

$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-2 \pm \sqrt{-4}}{2 \cdot 1} = \frac{-2 \pm 2i}{2} = -1 \pm 1i \Rightarrow \alpha = -1, \beta = 1$$

$$y = e^{\alpha t} \left(c_1 \cos(\beta t) + c_2 \sin(\beta t)\right)$$

$$y = e^{-t} \left(c_1 \cos(t) + c_2 \sin(t)\right)$$

---

$$\sin\left(\frac{\pi}{4}\right) = \frac{\sqrt{2}}{2} \qquad \cos\left(\frac{\pi}{4}\right) = \frac{\sqrt{2}}{2}$$

---

$$y\left(\frac{\pi}{4}\right) = e^{-\frac{\pi}{4}} \left(c_1 \cos\left(\frac{\pi}{4}\right) + c_2 \sin\left(\frac{\pi}{4}\right)\right) = e^{-\frac{\pi}{4}} \left(c_1 \cdot \frac{\sqrt{2}}{2} + c_2 \cdot \frac{\sqrt{2}}{2}\right)$$

$$y\left(\frac{\pi}{4}\right) = (c_1 + c_2) \left(e^{-\frac{\pi}{4}} \cdot \frac{\sqrt{2}}{2}\right) = 2$$

$$c_1 + c_2 = \frac{4}{e^{-\frac{\pi}{4}} \sqrt{2}} = \frac{4 \sqrt{2}}{e^{-\frac{\pi}{4}} \cdot 2} = 2 \sqrt{2} e^{\frac{\pi}{4}}$$

---

$$y' = c_1 \left[\left(e^{-t}\right) (-\sin(t)) + \left(-e^{-t}\right) (\cos(t))\right] + c_2 \left[\left(e^{-t}\right) (\cos(t)) + \left(-e^{-t}\right) (\sin(t))\right]$$

$$y' = e^{-t} \left[c_1 (-\sin(t) - \cos(t)) + c_2 (-\sin(t) + \cos(t))\right]$$

$$y'\left(\frac{\pi}{4}\right) = e^{-\frac{\pi}{4}} \left[c_1 \left(-\frac{\sqrt{2}}{2} - \frac{\sqrt{2}}{2}\right) + c_2 \left(-\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}\right)\right]$$

$$y'\left(\frac{\pi}{4}\right) = c_1 \left(-e^{-\frac{\pi}{4}} \sqrt{2}\right) = -2 \Rightarrow c_1 = \frac{2}{e^{-\frac{\pi}{4}} \sqrt{2}} = \frac{2 \sqrt{2}}{e^{-\frac{\pi}{4}} \cdot 2} = \sqrt{2} e^{\frac{\pi}{4}}$$

$$c_2 = 2 \sqrt{2} e^{\frac{\pi}{4}} - c_1 = \sqrt{2} e^{\frac{\pi}{4}}$$

---

$$y = \left(e^{-t}\right) \left(\sqrt{2} e^{\frac{\pi}{4}}\right) (\cos(t) + \sin(t))$$

$$
\boxed{
y = \left(\sqrt{2} e^{\frac{\pi}{4} - t}\right) (\cos(t) + \sin(t))
}
$$

## b
$$y'' - 2y' - 3y = 3te^{2t} \qquad y(0) = 1 \qquad y'(0) = 0$$

$$r^2 - 2r - 3 = 0$$

$$\Delta = b^2 - 4ac = (-2)^2 - 4 \cdot 1 \cdot (-3) = 4 + 12 = 16$$

$$\frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-(-2) \pm \sqrt{16}}{2 \cdot 1} = \frac{2 \pm 4}{2} = 1 \pm 2 = 3, -1$$

$$y_H = c_1 e^{3t} + c_2 e^{-t}$$

---

$$y_P = (At + B) e^{2t} = Ae^{2t}t + Be^{2t}$$

$$y'_P = A \left[ (e^{2t})(1) + (2e^{2t})(t) \right] + 2Be^{2t} = (A + 2At + 2B) e^{2t}$$

$$y'_P = Ae^{2t} + 2Ae^{2t}t + 2Be^{2t}$$

$$y''_P = 2Ae^{2t} + 2A \left[ (e^{2t})(1) + (2e^{2t})(t) \right] + 4Be^{2t}$$

$$y''_P = (2A + 2A + 4At + 4B) e^{2t} = (4A + 4At + 4B) e^{2t}$$

$$\left[ (4A + 4At + 4B) e^{2t} \right] - 2 \left[ (A + 2At + 2B) e^{2t} \right] - 3 \left[ (At + B) e^{2t} \right] = (3t) e^{2t}$$

$$4A + 4At + 4B - 2A - 4At - 4B - 3At - 3B = 3t$$

$$(4A - 4A - 3A)t + (4A + 4B - 2A - 4B - 3B) = 3t + 0$$

$$(-3A)t + (2A - 3B) = 3t + 0$$

$$
\begin{cases}
-3A = 3 \\
2A - 3B = 0
\end{cases}
$$

$$A = -1$$

$$B = \frac{2}{3} A = \frac{2}{3} (-1) = -\frac{2}{3}$$

$$y_P = \left( -t - \frac{2}{3} \right) e^{2t}$$

---

$$y = c_1 e^{3t} + c_2 e^{-t} + \left( -t - \frac{2}{3} \right) e^{2t} = c_1 e^{3t} + c_2 e^{-t} - te^{2t} - \frac{2}{3} e^{2t}$$

$$y' = 3c_1 e^{3t} - c_2 e^{-t} - \left[ (t)(2e^{2t}) + (1)(e^{2t}) \right] - \frac{4}{3} e^{2t}$$

$$y' = 3c_1 e^{3t} - c_2 e^{-t} - 2te^{2t} - e^{2t} - \frac{4}{3} e^{2t}$$

$$y(0) = c_1 + c_2 - \frac{2}{3} = 1 \Rightarrow c_1 + c_2 = \frac{5}{3}$$

$$y'(0) = 3c_1 - c_2 - 1 - \frac{4}{3} = 3c_1 - c_2 - \frac{7}{3} = 0 \Rightarrow 3c_1 - c_2 = \frac{7}{3}$$

$$
\begin{cases}
c_1 + c_2 = \frac{5}{3} \\
3c_1 - c_2 = \frac{7}{3}
\end{cases}
$$

$$4c_1 = \frac{12}{3} \Rightarrow c_1 = 1$$

$$c_2 = \frac{5}{3} - c_1 = \frac{2}{3}$$

$$\boxed{
y = e^{3t} + \frac{2}{3} e^{-t} + \left( -t - \frac{2}{3} \right) e^{2t}
}
$$

# Answer

$$
\boxed{

}
$$

## b
$$\boxed{

}
$$
