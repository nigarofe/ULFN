# Proposition

a) $ty' + 2y = 4t^2 \quad y(1) = 2$

b) $\frac{dy}{dx} = \frac{x^2}{1-y^2}$

c) $\frac{dy}{dx} = \frac{3x^2 + 4x + 2}{2(y-1)} \quad y(0) = -1$

d) $y' + y = e^t \sin(2t)$


# Step-by-step
## a
$$t y' + 2y = 4t^2 \qquad y(1) = 2$$

$$y' + \frac{2}{t} y = 4t$$

$$y' + p(t)y = g(t) \implies y = \frac{1}{\mu} (\int \mu g dt + c) \implies \mu = e^{\int p dt}$$

$$\mu = e^{\int 2t^{-1} dt} = e^{2 \ln |t|} = e^{\ln(t^2)} = t^2$$

$$y = \frac{1}{t^2} \left(\int (t^2)(4t) dt + c \right) = \frac{1}{t^2} \left(4 \int t^3 dt + c \right)$$

$$y = \frac{1}{t^2} (t^4 + c) = t^2 + ct^{-2}$$

$$y(1) = 1^2 + C(1)^{-2} = 1 + C = 2 \implies C = 1$$

$$y(t) = t^2 + t^{-2}$$

## b
$$\frac{dy}{dx} = \frac{x^2}{1-y^2}$$

$$(1-y^2) dy = x^2 dx$$

$$\int 1 - y^2 dy = \int x^2 dx$$

$$y - \frac{y^3}{3} = \frac{x^3}{3} + C$$

$$-y^3 + 3y - x^3 = C$$

$$y^3 - 3y + x^3 = C$$

## c
$$\frac{dy}{dx} = \frac{3x^2+4x+2}{2(y-1)} \qquad y(0)=-1$$

$$\int (2y-2) dy = \int 3x^2+4x+2 dx$$

$$y^2-2y = x^3+2x^2+2x+c$$

$$(y-1)^2-1 = x^3+2x^2+2x+c$$

$$y-1 = \pm\sqrt{x^3+2x^2+2x+c+1}$$

$$y = \pm\sqrt{x^3+2x^2+2x+c+1} + 1$$

$$y(0) = \pm\sqrt{c+1} + 1 = -1$$

$$\pm\sqrt{c+1} = -2 \Rightarrow c=3$$

$$y(x) = -\sqrt{x^3+2x^2+2x+4} + 1$$

## d
$$y' + y = e^t \sin(2t)$$
$$y' + p(t) y = g(t) \qquad y = \frac{1}{\mu} (\int \mu g \, dt + c) \qquad \mu = e^{\int p \, dt}$$
$$p(t) = 1 \implies \mu = e^{\int 1 dt} = e^t$$
$$y = \frac{1}{e^t} \left(\int (e^t)(e^t \sin(2t)) dt + c \right) = \frac{1}{e^t} \left({\color{green}\int e^{2t} \sin(2t) dt} + c \right)$$

---

$$\color{green}\int e^{2t} \sin(2t) dt$$
$$\int uv' = uv - \int u'v$$
$$u = \sin(2t) \qquad u' = 2 \cos(2t)$$
$$v' = e^{2t} \qquad v = \frac{1}{2} e^{2t}$$
$$\int e^{2t} \sin(2t) dt = (\sin(2t)) \left(\frac{1}{2} e^{2t}\right) - \int (2 \cos(2t)) \left(\frac{1}{2} e^{2t}\right) dt$$
$$= {\color{blue} \frac{1}{2} e^{2t} \sin(2t) -} {\color{red}\int e^{2t} \cos(2t) dt}$$

---

$${\color{red}\int e^{2t} \cos(2t) dt}$$
$$\int uv' = uv - \int u'v$$
$$u = \cos(2t) \qquad u' = -2 \sin(2t)$$
$$v' = e^{2t} \qquad v = \frac{1}{2} e^{2t}$$
$$\int e^{2t} \cos(2t) dt = (\cos(2t)) \left(\frac{1}{2} e^{2t}\right) - \int (-2 \sin(2t)) \left(\frac{1}{2} e^{2t}\right) dt$$
$$= \color{red} \frac{1}{2} e^{2t} \cos(2t) + \int e^{2t} \sin(2t) dt$$

---

$${\color{green}\int e^{2t} \sin(2t) dt} = {\color{blue} \frac{1}{2} e^{2t} \sin(2t) -} \color{red} \left(\frac{1}{2} e^{2t} \cos(2t) + \int e^{2t} \sin(2t) dt\right)$$
$$2 \int e^{2t} \sin(2t) dt = \frac{1}{2} e^{2t} (\sin(2t) - \cos(2t))$$
$$\int e^{2t} \sin(2t) dt = \frac{1}{4} e^{2t} (\sin(2t) - \cos(2t))$$
    
---

$$y = \frac{1}{e^t} \left(\frac{1}{4} e^{2t} (\sin(2t) - \cos(2t)) + c \right)$$
$$y = \frac{1}{4} e^t (\sin(2t) - \cos(2t)) + c e^{-t}$$

# Answer
a) $y(t) = t^2 + t^{-2}$

b) $y^3 - 3y + x^3 = C$

c) $y(x) = -\sqrt{x^3+2x^2+2x+4} + 1$

d) $y(t) = \frac{1}{4} e^t (\sin(2t) - \cos(2t)) + c e^{-t}$
