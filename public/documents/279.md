# Proposition
Calcule a transformada de Laplace das seguintes funções:

a) $f(t) = 1$ para $t \geq 0$

b) $f(t) = e^{at}$ para $t \geq 0$

c) $\begin{cases}
    f(t) = 1 & \text{se } 0 \leq t < 1 \\
    f(t) = k & \text{se } t = 1 \\
    f(t) = 0 & \text{se } t > 1
\end{cases}$

d) $f(t) = \sin(at)$ para $t \geq 0$

e) $f(t) = 5 e^{-2t} - 3 \sin(4t)$ para $t \geq 0$

Calcule $y(t)$ usando a transformada de Laplace para a seguinte equação diferencial:

f) $y'' - y' - 2y = 0 \qquad y(0) = 1 \qquad y'(0) = 0$

# Step-by-step

## f
$Y(s) = \frac{s + 1}{s^2 - s - 2}$

# Answer

a) $\boxed{\mathcal{L}\{1\} = \frac{1}{s}}$

b) $\boxed{\mathcal{L}\{e^{at}\} = \frac{1}{s-a}}$

c) $\boxed{\mathcal{L}\{f(t)\} = \frac{1}{s} (1 - e^{-s})}$

d) $\boxed{\mathcal{L}\{\sin(at)\} = \frac{a}{s^2 + a^2}}$

e) $\boxed{\mathcal{L}\{5 e^{-2t} - 3 \sin(4t)\} = \frac{5}{s + 2} - \frac{12}{s^2 + 16}}$

f) $\boxed{y(t) = \frac{1}{3} e^{2t} + \frac{2}{3} e^{-t}}$
