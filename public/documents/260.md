# Proposition
a) Suponha que uma mola tenha uma massa $m$ e constante de mola $k$ e seja $\omega = \sqrt{k/m}$. Suponha uma constante de amortecimento tão pequena que a força de amortecimento seja desprezível. Se uma força externa $F(t) = F_0 \cos \omega_0 t$ for aplicada, onde $\omega_0 \neq \omega$, use o método dos coeficientes indeterminados para mostrar que o movimento da massa é descrito pela equação

$$x(t) = c_1 \cos \omega t + c_2 \sin \omega t + \frac{F_0}{m(\omega^2 - \omega_0^2)} \cos \omega_0 t \quad (1)$$

b) Como na letra a), considere uma mola com uma massa $m$, constante da mola $k$ e constante de amortecimento $c = 0$, e seja $\omega = \sqrt{k/m}$. Se uma força externa $F(t) = F_0 \cos \omega t$ for aplicada (a frequência aplicada é igual à frequência natural), use o método dos coeficientes indeterminados para mostrar que o movimento da massa é dado por

$$x(t) = c_1 \cos \omega t + c_2 \sin \omega t + \frac{F_0}{2m\omega} t \sin \omega t$$

c) Mostre que se $\omega_0 \neq \omega$, mas $\omega/\omega_0$ é um número racional, então o movimento descrito pela Equação (1) é periódico.

# Notes


# Step-by-step
## Gemini

### a) Demonstração do movimento para $\omega_0 \neq \omega$

A equação diferencial que descreve o movimento de uma massa em uma mola sem amortecimento ($c=0$), sujeita a uma força externa $F(t) = F_0 \cos \omega_0 t$, é dada pela Segunda Lei de Newton:

$$mx'' + kx = F_0 \cos \omega_0 t$$

Dividindo toda a equação por $m$ e lembrando que a frequência natural é $\omega = \sqrt{k/m}$ (o que implica $\omega^2 = k/m$), temos:

$$x'' + \omega^2 x = \frac{F_0}{m} \cos \omega_0 t$$

A solução geral desta equação linear não homogênea é a soma da solução homogênea $x_h(t)$ com uma solução particular $x_p(t)$.

**1. Solução Homogênea ($x_h$):**
A equação homogênea associada é $x'' + \omega^2 x = 0$. A equação característica é $r^2 + \omega^2 = 0$, que tem raízes complexas $r = \pm i\omega$. Assim, a solução homogênea é:


$$x_h(t) = c_1 \cos \omega t + c_2 \sin \omega t$$

**2. Solução Particular ($x_p$):**
Pelo método dos coeficientes indeterminados, como o termo não homogêneo é $\frac{F_0}{m} \cos \omega_0 t$ e sabemos que $\omega_0 \neq \omega$, propomos a seguinte solução particular:


$$x_p(t) = A \cos \omega_0 t + B \sin \omega_0 t$$

Calculando a primeira e segunda derivadas de $x_p$:


$$x_p'(t) = -A\omega_0 \sin \omega_0 t + B\omega_0 \cos \omega_0 t$$

$$x_p''(t) = -A\omega_0^2 \cos \omega_0 t - B\omega_0^2 \sin \omega_0 t$$

Substituindo $x_p$ e $x_p''$ na equação diferencial original:


$$(-A\omega_0^2 \cos \omega_0 t - B\omega_0^2 \sin \omega_0 t) + \omega^2 (A \cos \omega_0 t + B \sin \omega_0 t) = \frac{F_0}{m} \cos \omega_0 t$$

Agrupando os termos de cosseno e seno:


$$A(\omega^2 - \omega_0^2) \cos \omega_0 t + B(\omega^2 - \omega_0^2) \sin \omega_0 t = \frac{F_0}{m} \cos \omega_0 t$$

Igualando os coeficientes de ambos os lados:
Para o seno: $B(\omega^2 - \omega_0^2) = 0 \implies B = 0$ (pois $\omega \neq \omega_0$).
Para o cosseno: $A(\omega^2 - \omega_0^2) = \frac{F_0}{m} \implies A = \frac{F_0}{m(\omega^2 - \omega_0^2)}$

Portanto, a solução particular é $x_p(t) = \frac{F_0}{m(\omega^2 - \omega_0^2)} \cos \omega_0 t$.

A solução geral $x(t) = x_h(t) + x_p(t)$ é, finalmente:


$$x(t) = c_1 \cos \omega t + c_2 \sin \omega t + \frac{F_0}{m(\omega^2 - \omega_0^2)} \cos \omega_0 t$$


### b) Demonstração do movimento para $\omega_0 = \omega$ (Ressonância)

Agora, a frequência da força aplicada é igual à frequência natural do sistema. A equação diferencial se torna:

$$x'' + \omega^2 x = \frac{F_0}{m} \cos \omega t$$

A solução homogênea permanece a mesma:


$$x_h(t) = c_1 \cos \omega t + c_2 \sin \omega t$$

**Solução Particular ($x_p$):**
A função $\cos \omega t$ do lado direito já faz parte da solução homogênea. Para aplicar o método dos coeficientes indeterminados neste caso (onde há multiplicidade), devemos multiplicar a nossa "suposição" inicial por $t$. Propomos:


$$x_p(t) = t(A \cos \omega t + B \sin \omega t)$$

Calculando as derivadas usando a regra do produto:


$$x_p'(t) = (A \cos \omega t + B \sin \omega t) + t(-A\omega \sin \omega t + B\omega \cos \omega t)$$

$$x_p''(t) = (-A\omega \sin \omega t + B\omega \cos \omega t) + (-A\omega \sin \omega t + B\omega \cos \omega t) + t(-A\omega^2 \cos \omega t - B\omega^2 \sin \omega t)$$

Simplificando $x_p''$:


$$x_p''(t) = -2A\omega \sin \omega t + 2B\omega \cos \omega t - \omega^2 t(A \cos \omega t + B \sin \omega t)$$

Note que o último termo é exatamente $-\omega^2 x_p(t)$. Substituindo $x_p''$ e $x_p$ na equação diferencial $x_p'' + \omega^2 x_p = \frac{F_0}{m} \cos \omega t$:


$$[-2A\omega \sin \omega t + 2B\omega \cos \omega t - \omega^2 x_p(t)] + \omega^2 x_p(t) = \frac{F_0}{m} \cos \omega t$$

Os termos com $x_p(t)$ se cancelam, restando:


$$-2A\omega \sin \omega t + 2B\omega \cos \omega t = \frac{F_0}{m} \cos \omega t$$

Igualando os coeficientes:
Para o seno: $-2A\omega = 0 \implies A = 0$
Para o cosseno: $2B\omega = \frac{F_0}{m} \implies B = \frac{F_0}{2m\omega}$

Logo, a solução particular é:


$$x_p(t) = \frac{F_0}{2m\omega} t \sin \omega t$$

Somando com a solução homogênea, obtemos a equação de movimento:


$$x(t) = c_1 \cos \omega t + c_2 \sin \omega t + \frac{F_0}{2m\omega} t \sin \omega t$$


*(Nota: O termo "t" multiplicando a função seno indica que a amplitude oscilará e crescerá linearmente para o infinito ao longo do tempo, o que caracteriza a ressonância pura).*

### c) Prova de que o movimento é periódico se $\omega/\omega_0$ é racional

A Equação (1) descreve o movimento como uma soma de duas funções oscilatórias:


$$x(t) = \underbrace{c_1 \cos \omega t + c_2 \sin \omega t}_{x_h(t)} + \underbrace{\frac{F_0}{m(\omega^2 - \omega_0^2)} \cos \omega_0 t}_{x_p(t)}$$

Para provar que $x(t)$ é periódica, precisamos mostrar que existe um período $T > 0$ tal que $x(t + T) = x(t)$ para todo $t$.
Sabemos que a soma de duas funções periódicas é periódica se, e somente se, a razão entre seus períodos é um número racional.

1. O período da parte homogênea $x_h(t)$ (que tem frequência angular $\omega$) é $T_1 = \frac{2\pi}{\omega}$.
2. O período da parte particular $x_p(t)$ (que tem frequência angular $\omega_0$) é $T_2 = \frac{2\pi}{\omega_0}$.

Vamos calcular a razão entre os períodos:


$$\frac{T_1}{T_2} = \frac{\frac{2\pi}{\omega}}{\frac{2\pi}{\omega_0}} = \frac{\omega_0}{\omega}$$

O problema afirma que a razão $\frac{\omega}{\omega_0}$ é um número racional. Se $\frac{\omega}{\omega_0}$ é um número racional (digamos $p/q$, onde $p$ e $q$ são inteiros não nulos), então sua inversa $\frac{\omega_0}{\omega} = \frac{q}{p}$ também é um número racional.

Assim, temos:


$$\frac{T_1}{T_2} = \frac{q}{p} \implies p T_1 = q T_2$$

Seja $T = p T_1 = q T_2$. Este valor $T$ é um múltiplo inteiro de ambos os períodos $T_1$ e $T_2$.
Portanto, após um tempo $T$, ambas as parcelas da equação $x(t)$ terão completado um número inteiro de ciclos ($p$ ciclos para $x_h$ e $q$ ciclos para $x_p$). Concluímos que:


$$x(t + T) = x_h(t + pT_1) + x_p(t + qT_2) = x_h(t) + x_p(t) = x(t)$$

Como $x(t + T) = x(t)$, o movimento descrito pela Equação (1) é estritamente periódico, com período fundamental sendo o mínimo múltiplo comum de $T_1$ e $T_2$.
