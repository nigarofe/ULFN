# Proposition

a) $y'' + y = \sin t$

b) $y'' + y = \tan x \qquad 0 < x < \dfrac{\pi}{2}$

## MĂ©todo da variaĂ§ĂŁo de parĂąmetros
$$ay'' + by' + cy = G(t)$$

$$y = y_H + y_P$$

---

$$y_H = c_1 y_1(t) + c_2 y_2(t)$$

$$y_P = u_1(t) y_1(t) + u_2(t) y_2(t)$$

$$y'_P = u_1' y_1 + u_1 y_1' + u_2' y_2 + u_2 y_2'$$

$$y'_P = (u_1' y_1 + u_2' y_2) + (u_1 y_1' + u_2 y_2')$$

---

Vamos impor que as derivadas de $u_1$ e $u_2$ satisfaĂ§am a seguinte condiĂ§ĂŁo:

$$u_1' y_1 + u_2' y_2 = 0 \tag{1}$$

---

$$y'_P = u_1 y_1' + u_2 y_2'$$

$$y_P'' = u_1' y_1' + u_1 y_1'' + u_2' y_2' + u_2 y_2''$$

Substituindo na EDO:

$$a \left[ u_1' y_1' + u_1 y_1'' + u_2' y_2' + u_2 y_2'' \right] + b \left[ u_1 y_1' + u_2 y_2' \right] + c \left[ u_1 y_1 + u_2 y_2 \right] = G(t)$$

$$u_1 (a y_1'' + b y_1' + c y_1) + u_2 (a y_2'' + b y_2' + c y_2) + a (u_1' y_1' + u_2' y_2') = G(t)$$

Como $y_1$ e $y_2$ sĂŁo soluĂ§Ă”es da equaĂ§ĂŁo homogĂȘnea, temos:

$$u_1 \underbrace{(a y_1'' + b y_1' + c y_1)}_{= 0} + u_2 \underbrace{(a y_2'' + b y_2' + c y_2)}_{= 0} + a (u_1' y_1' + u_2' y_2') = G(t)$$

$$a (u_1' y_1' + u_2' y_2') = G(t) \tag{2}$$

---

Montando um sistema com as equaĂ§Ă”es (1) e (2):

$$\begin{cases}
u_1' y_1 + u_2' y_2 = 0 \\
u_1' y_1' + u_2' y_2' = \dfrac{G(t)}{a}
\end{cases}$$

ApĂłs achar $u_1'$ e $u_2'$, basta integrar para encontrar $u_1$ e $u_2$, e entĂŁo substituir na expressĂŁo de $y_P$.



# Step-by-step
## a
$$y'' + y = \sin(t)$$
$$r^2 + 1 = 0$$
$$r = \pm \sqrt{-1} = 0 \pm 1i \Rightarrow \alpha = 0, \beta = 1$$
$$y_H = e^{\alpha t} (c_1 \cos(\beta t) + c_2 \sin(\beta t))$$
$$y_H = c_1 \cos(t) + c_2 \sin(t)$$
$$y_P = A t \cos(t) + B t \sin(t)$$
$$y'_P = A (-t \sin(t) + 1 \cos(t)) + B (t \cos(t) + 1 \sin(t))$$
$$y'_P = -A t \sin(t) + A \cos(t) + B t \cos(t) + B \sin(t)$$
$$y''_P = -A (t \cos(t) + 1 \cdot \sin(t)) - A \sin(t) + B (-t \sin(t) + 1 \cdot \cos(t)) + B \cos(t)$$
$$y''_P = -A t \cos(t) - 2A \sin(t) - B t \sin(t) + 2B \cos(t)$$

Substituindo na EDO e cancelando alguns termos:
$$-2A \sin(t) + 2B \cos(t) = \sin(t)$$

$$\begin{cases} -2A = 1 \\ 2B = 0 \end{cases} \Rightarrow A = -\frac{1}{2}, \quad B = 0$$

$$y_P = -\frac{1}{2} t \cos(t)$$
$$y = c_1 \cos(t) + c_2 \sin(t) - \frac{1}{2} t \cos(t)$$

## b
$$y'' + y = \tan(x) \quad 0 < x < \frac{\pi}{2}$$

$$r^2 + 1 = 0$$

$$r = \pm \sqrt{-1} = 0 \pm i \implies \alpha = 0 ; \beta = 1$$

$$y_H = e^{\alpha x} \left( c_1 \cos(\beta x) + c_2 \sin(\beta x) \right) = c_1 \cos(x) + c_2 \sin(x)$$

Pelo mĂ©todo da variaĂ§ĂŁo de parĂąmetros

$$y_H = c_1 y_1 + c_2 y_2$$

$$y_P = u_1 y_1 + u_2 y_2$$

$$y_P' = (u_1' y_1 + u_1 y_1') + (u_2' y_2 + u_2 y_2') = u_1' y_1 + u_2' y_2 + u_1 y_1' + u_2 y_2'$$

Impondo que $u_1' y_1 + u_2' y_2 = 0$

$$y_P' = u_1 y_1' + u_2 y_2'$$

$$y_P'' = (u_1' y_1' + u_1 y_1'') + (u_2' y_2' + u_2 y_2'')$$

$$y_1 = \cos(x) \quad y_1' = -\sin(x) \quad y_1'' = -\cos(x)$$

$$y_2 = \sin(x) \quad y_2' = \cos(x) \quad y_2'' = -\sin(x)$$

$$y_P = u_1 \cos(x) + u_2 \sin(x)$$

$$y_P' = -u_1 \sin(x) + u_2 \cos(x)$$

$$y_P'' = (-u_1' \sin(x) - u_1 \cos(x)) + (u_2' \cos(x) - u_2 \sin(x))$$

---

$$y_P'' + y_P = \tan(x)$$

$$-u_1' \sin(x) + u_2' \cos(x) = \tan(x)$$

$$
\begin{cases}
u_1' \cos(x) + u_2' \sin(x) = 0 \\
-u_1' \sin(x) + u_2' \cos(x) = \tan(x)
\end{cases}
$$

$$u_1' = - \frac{u_2' \sin(x)}{\cos(x)}$$

$$- \left( - \frac{u_2' \sin(x)}{\cos(x)} \right) \sin(x) + u_2' \cos(x) = \tan(x)$$

$$u_2' \left( \frac{\sin^2(x)}{\cos(x)} + \cos(x) \right) = \tan(x)$$

$$u_2' \left( \frac{\sin^2(x) + \cos^2(x)}{\cos(x)} \right) = \frac{\sin(x)}{\cos(x)}$$

$$u_2' \left( \frac{1}{\cos(x)} \right) = \frac{\sin(x)}{\cos(x)}$$

$$u_2' = \sin(x)$$

---

$$u_1' = - \frac{\sin(x) \sin(x)}{\cos(x)} = - \frac{\sin^2(x)}{\cos(x)} = + \frac{\cos^2(x) - 1}{\cos(x)}$$

$$u_1' = \frac{\cos^2(x)}{\cos(x)} - \frac{1}{\cos(x)} = \cos(x) - \sec(x)$$

$$\int u_1' dx = \int \cos(x) - \sec(x) dx = \sin(x) - \ln|\sec(x) + \tan(x)|$$

$$\int u_2' dx = \int \sin(x) dx = -\cos(x)$$

$$y_P = (\sin(x) - \ln|\sec(x) + \tan(x)|) \cos(x) + (-\cos(x)) \sin(x)$$

$$y = y_H + y_P$$

$$\boxed{y = c_1 \cos(x) + c_2 \sin(x) + (\sin(x) - \ln|\sec(x) + \tan(x)|) \cos(x) - \cos(x) \sin(x)}$$


# Answer
## a
$$
\boxed{
y = c_1 \cos t + c_2 \sin t - \frac{1}{2} t \cos t
}
$$

## b
$$\boxed{y = c_1 \cos(x) + c_2 \sin(x) + (\sin(x) - \ln|\sec(x) + \tan(x)|) \cos(x) - \cos(x) \sin(x)}$$
